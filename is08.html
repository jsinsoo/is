6. 학습할 코드 1개

계단 오르기 문제에서 전체 문제와 부분 문제의 관계 구현하기

def count_ways(n):
    """1칸 또는 2칸씩 올라 n번째 계단에 도달하는 방법 수"""
    if n <= 0:
        return 0

    if n == 1:
        return 1

    if n == 2:
        return 2

    ways = [0] * (n + 1)
    ways[1] = 1
    ways[2] = 2

    for step in range(3, n + 1):
        ways[step] = ways[step - 1] + ways[step - 2]

    return ways[n]


n = int(input("계단 수를 입력하세요: "))
print("올라가는 방법 수:", count_ways(n))

상세 설명

코드 요소설명
count_ways(n)n번째 계단에 도달하는 방법 수를 구하는 전체 문제를 함수로 표현한다.
n == 1, n == 2더 이상 분해하지 않고 바로 답을 알 수 있는 기저 문제이다.
ways = [0] * (n + 1)각 작은 계단 문제의 답을 저장할 표를 만든다.
ways[step - 1]마지막에 1칸 올라온 경우의 수에 해당하는 부분 문제이다.
ways[step - 2]마지막에 2칸 올라온 경우의 수에 해당하는 부분 문제이다.
ways[step] = ...두 부분 문제의 결과를 더해 현재 전체 문제의 답을 만든다.
반복문작은 문제의 답부터 차례대로 계산하여 더 큰 문제의 답을 만든다.
return ways[n]모든 부분 문제 계산이 끝난 뒤 전체 문제의 답을 반환한다.

예: 4번째 계단에 도달하는 방법은 3번째 계단에서 1칸 오는 경우 3가지와 2번째 계단에서 2칸 오는 경우 2가지를 합한 5가지이다.
