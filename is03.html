6. 학습할 코드 1개

팩토리얼을 반복구조와 재귀구조로 비교하기

def factorial_iterative(n):
    """반복구조를 이용해 n!을 계산한다."""
    result = 1
    for number in range(2, n + 1):
        result *= number
    return result


def factorial_recursive(n):
    """재귀구조를 이용해 n!을 계산한다."""
    if n <= 1:                 # 기저 조건
        return 1
    return n * factorial_recursive(n - 1)  # 재귀관계


n = int(input("0 이상의 정수를 입력하세요: "))

if n < 0:
    print("음수의 팩토리얼은 이 프로그램에서 계산하지 않습니다.")
else:
    iterative_result = factorial_iterative(n)
    recursive_result = factorial_recursive(n)

    print("반복 결과:", iterative_result)
    print("재귀 결과:", recursive_result)
    print("두 결과가 같은가?", iterative_result == recursive_result)

상세 설명

코드 요소설명
result = 1곱셈의 항등원 1에서 누적을 시작한다. 0!도 1이므로 초기값으로 적절하다.
range(2, n + 1)2부터 n까지 차례로 순회한다. 끝값 n을 포함하기 위해 n+1을 사용한다.
result *= number현재 누적값에 number를 곱해 상태를 갱신한다.
if n <= 1재귀 호출을 멈추는 기저 조건이다. 이것이 없으면 호출이 계속된다.
n * factorial_recursive(n - 1)n! = n × (n-1)!이라는 재귀관계를 코드로 표현한다.
결과 비교식서로 다른 구현이 같은 결과를 내는지 확인하여 정확성을 검증한다.

예: n=5이면 반복형은 1→2→6→24→120으로 누적하고, 재귀형은 5×4!→5×4×3!→…→120으로 되돌아오며 계산한다.
