6. 학습할 코드 1개

portfolio = {
    "title": "교실 공기질 알림 시스템",
    "problem": "이산화탄소 농도 상승을 즉시 알기 어려웠다.",
    "achievement": "센서 데이터를 분석해 환기 필요 상태를 표시했다.",
    "evidence": [
        "분석 코드",
        "테스트 결과",
        "사용자 피드백",
        "개선 전후 비교표"
    ],
    "strength": "문제를 데이터와 알고리즘으로 구조화했다.",
    "limitation": "한 교실의 짧은 기간 자료만 사용했다.",
    "next_goal": "여러 교실과 계절 자료로 분석 범위를 넓힌다."
}


def create_portfolio_report(data):
    report = []

    report.append(f'프로젝트: {data["title"]}')
    report.append(f'문제: {data["problem"]}')
    report.append(f'주요 성과: {data["achievement"]}')

    report.append("근거 자료")
    for index, item in enumerate(data["evidence"], start=1):
        report.append(f'{index}. {item}')

    report.append(f'강점: {data["strength"]}')
    report.append(f'한계: {data["limitation"]}')
    report.append(f'다음 목표: {data["next_goal"]}')

    return "\n".join(report)


result = create_portfolio_report(portfolio)
print(result)
코드 요소상세 설명
portfolio프로젝트 주제, 문제, 성과, 근거, 강점, 한계, 다음 목표를 저장한 딕셔너리이다.
evidence자기평가를 뒷받침하는 코드, 테스트, 피드백, 비교표 등의 목록이다.
create_portfolio_report포트폴리오 데이터를 읽어 구조화된 성찰 보고서를 생성한다.
report보고서에 포함할 문장을 순서대로 저장하는 리스트이다.
enumerate(..., start=1)근거 자료에 1번부터 번호를 붙인다.
"\n".join(report)여러 문장을 줄바꿈으로 연결하여 하나의 보고서 문자열로 만든다.

확장 활동: 담당 역할, 사용한 오픈소스, 윤리적 고려, 피드백 반영 내용, 버전별 변경 기록을 추가해 완전한 학습 포트폴리오로 발전시킨다.
