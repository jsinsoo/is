101~115if, elif, else 조건문

1. 조건문 — 예제 101~125

예제 101. 기본 if문

if조건문 
score = 90

if score >= 80:
print("합격입니다.")
실행 결과
합격입니다.
설명
if 뒤의 조건이 참일 때만 들여쓰기된 코드가 실행됩니다.

예제 102. 조건이 거짓인 경우

print("합격입니다.")

print("프로그램이 종료되었습니다.")
실행 결과
프로그램이 종료되었습니다.
설명
조건이 거짓이면 if문 내부는 실행되지 않고 다음 코드로 넘어갑니다.

4. 제어문 — 예제 146~150

예제 146. break 사용

for number in range(1, 11):
    if number == 6:
        break
    print(number)
실행 결과
1
2
3
4
5
설명
break는 반복문을 즉시 종료합니다.

예제 147. continue 사용

for number in range(1, 11):
    if number % 2 != 0:
        continue
    print(number)
실행 결과
2
4
6
8
10
설명
continue는 현재 반복만 건너뛰고 다음 반복으로 넘어갑니다.

예제 148. pass 사용

for number in range(3):
    pass

print("반복문이 끝났습니다.")
실행 결과
반복문이 끝났습니다.
설명
pass는 아직 작성할 코드가 없을 때 문법 오류를 막기 위해 사용하는 빈 명령입니다.

예제 149. for-else문

numbers = [1, 3, 5, 7]

for number in numbers:
if number % 2 == 0:
print("짝수를 찾았습니다.")
break
else:
print("짝수가 없습니다.")
실행 결과
짝수가 없습니다.
설명
반복문이 break 없이 정상 종료되면 else 부분이 실행됩니다.

예제 150. 특정 값 찾기

students = ["민수", "지민", "서연"]
target = "지민"

for student in students:
if student == target:
print("학생을 찾았습니다.")
break
else:
print("학생을 찾지 못했습니다.")
실행 결과
학생을 찾았습니다.
설명
반복문과 조건문을 결합하면 리스트에서 특정 데이터를 찾는 기본 탐색을 만들 수 있습니다.

3편 핵심 요약

  • if, elif, else는 조건에 따라 다른 코드를 실행합니다.
  • for문은 리스트, 문자열, range 등의 요소를 순서대로 반복합니다.
  • while문은 조건이 참인 동안 반복합니다.
  • break는 반복 종료, continue는 현재 반복 건너뛰기, pass는 빈 코드 블록 작성에 사용합니다.
  • 조건문과 반복문을 결합하면 메뉴, 게임, 점수 처리, 탐색 프로그램을 만들 수 있습니다.
