🧩 학습할 핵심 개념 10개

1 빈도와 빈도표

몇 번 나타났는가

개념 상세 설명: 빈도는 특정 값이나 범주가 자료에서 나타난 횟수이다. 빈도표는 범주별 개수를 정리하여 어떤 값이 많고 적은지 한눈에 보여 준다.

responses = ["찬성", "반대", "찬성", "보통", "찬성"]
frequency = {}

for item in responses:
    frequency[item] = frequency.get(item, 0) + 1

print(frequency)
코드 상세 설명: 딕셔너리의 get()을 이용하여 각 응답의 등장 횟수를 센다. 결과를 통해 가장 많은 응답과 범주별 차이를 확인할 수 있다.
[개념 1] 빈도에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ⑤ — 빈도는 범주별 등장 횟수를 나타낸다.

2 상대도수와 비율

전체 중 어느 정도인가

개념 상세 설명: 상대도수는 범주의 빈도를 전체 자료 수로 나눈 값이다. 표본 크기가 다른 집단을 비교할 때 단순 빈도보다 비율이 더 적절할 수 있다.

counts = {"A": 12, "B": 8, "C": 5}
total = sum(counts.values())

ratios = {
    key: round(value / total * 100, 1)
    for key, value in counts.items()
}
print(ratios)
코드 상세 설명: 각 범주의 빈도를 전체 빈도로 나눈 뒤 백분율로 바꾼다. 전체 크기가 다른 집단을 비교할 때 상대도수를 함께 제시하는 것이 좋다.
[개념 2] 상대도수에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ④ — 상대도수는 전체에서 차지하는 비율이다.

3 평균

값의 균형점

개념 상세 설명: 평균은 모든 값을 더한 뒤 자료 수로 나눈 값이다. 전체 값을 반영하지만 매우 크거나 작은 이상값의 영향을 크게 받을 수 있다.

scores = [78, 82, 85, 88, 92]
mean = sum(scores) / len(scores)
print("평균:", mean)
코드 상세 설명: sum()으로 총합을 구하고 len()으로 자료 수를 구해 평균을 계산한다. 결측값이 있을 때는 유효한 값만 사용해야 한다.
[개념 3] 평균에 대한 옳은 설명 3개를 고르시오.
정답: ②, ③, ⑤ — 평균은 전체 값을 반영하지만 이상값에 민감하다.

4 중앙값

가운데 위치의 값

개념 상세 설명: 중앙값은 자료를 크기순으로 정렬했을 때 가운데 위치한 값이다. 자료 수가 짝수이면 가운데 두 값의 평균을 사용한다. 이상값이 있는 자료에서 평균보다 안정적일 수 있다.

values = [12, 18, 20, 22, 95]
ordered = sorted(values)
middle = len(ordered) // 2
median = ordered[middle]

print("정렬:", ordered)
print("중앙값:", median)
코드 상세 설명: 자료를 정렬하고 가운데 인덱스의 값을 선택한다. 홀수 개 자료의 예이며, 짝수 개일 때는 가운데 두 값의 평균을 구해야 한다.
[개념 4] 중앙값에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ④ — 중앙값은 정렬된 자료의 가운데 위치를 나타낸다.

5 최빈값

가장 자주 나타난 값

개념 상세 설명: 최빈값은 가장 많이 나타난 값이나 범주이다. 범주형 자료에도 사용할 수 있으며, 최빈값이 여러 개이거나 존재하지 않을 수도 있다.

from collections import Counter

data = ["A", "B", "A", "C", "A", "B"]
counter = Counter(data)
mode = counter.most_common(1)[0]

print("빈도:", counter)
print("최빈값:", mode)
코드 상세 설명: Counter로 각 값의 빈도를 세고 most_common(1)로 가장 자주 나타난 값을 찾는다.
[개념 5] 최빈값에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ⑤ — 최빈값은 가장 빈번한 값이며 하나 이상일 수 있다.

6 최솟값·최댓값·범위

전체 폭 살피기

개념 상세 설명: 범위는 최댓값에서 최솟값을 뺀 값으로 자료가 퍼진 전체 폭을 나타낸다. 계산은 쉽지만 두 극단값에만 의존하므로 이상값에 매우 민감하다.

values = [15, 17, 19, 22, 28]
minimum = min(values)
maximum = max(values)
data_range = maximum - minimum

print("최솟값:", minimum)
print("최댓값:", maximum)
print("범위:", data_range)
코드 상세 설명: min()과 max()로 양끝값을 구하고 차이로 범위를 계산한다. 퍼짐을 더 안정적으로 보려면 사분위수도 함께 본다.
[개념 6] 범위에 대한 옳은 설명 3개를 고르시오.
정답: ②, ④, ⑤ — 범위는 전체 폭을 보여 주지만 극단값에 민감하다.

7 사분위수와 사분위범위

가운데 50%의 퍼짐

개념 상세 설명: 사분위수는 정렬된 자료를 네 부분으로 나누는 경계값이다. 제1사분위수와 제3사분위수의 차이인 사분위범위는 가운데 50% 자료의 퍼짐을 나타내며 이상값의 영향을 범위보다 덜 받는다.

values = [10, 12, 15, 18, 20, 22, 25, 30]
ordered = sorted(values)

q1 = (ordered[1] + ordered[2]) / 2
q3 = (ordered[5] + ordered[6]) / 2
iqr = q3 - q1

print("Q1:", q1, "Q3:", q3, "IQR:", iqr)
코드 상세 설명: 정렬된 자료의 아래쪽과 위쪽 절반에서 중앙값을 계산하여 Q1과 Q3를 구하고 차이로 IQR을 계산한다.
[개념 7] 사분위수와 사분위범위에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ⑤ — IQR은 가운데 절반의 퍼짐을 나타낸다.

8 분포의 모양과 치우침

값이 어디에 모였는가

개념 상세 설명: 분포는 값이 나타나는 전체적인 모양이다. 대칭 분포에서는 평균과 중앙값이 비슷할 수 있고, 큰 값 쪽 꼬리가 길면 평균이 중앙값보다 커지는 경향이 있다.

data = [10, 11, 11, 12, 13, 14, 40]
mean = sum(data) / len(data)
ordered = sorted(data)
median = ordered[len(data)//2]

print("평균:", round(mean, 2))
print("중앙값:", median)
코드 상세 설명: 큰 값 40 때문에 평균이 중앙값보다 커진다. 평균과 중앙값의 차이는 치우침과 이상값을 탐색하는 단서가 될 수 있다.
[개념 8] 분포의 모양에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ④ — 대표값 하나만으로 분포 전체를 설명할 수 없다.

9 집단별 요약 통계 비교

같은 기준으로 비교

개념 상세 설명: 두 집단을 비교할 때는 평균만 보지 않고 표본 수, 중앙값, 범위와 사분위범위도 함께 확인한다. 표본 크기와 수집 조건이 다르면 단순 비교에 주의해야 한다.

group_a = [18, 20, 21, 22, 24]
group_b = [15, 17, 21, 25, 27]

for name, data in [("A", group_a), ("B", group_b)]:
    mean = sum(data) / len(data)
    spread = max(data) - min(data)
    print(name, "평균:", mean, "범위:", spread)
코드 상세 설명: 두 집단의 평균과 범위를 같은 방식으로 계산한다. 평균이 비슷해도 퍼짐이 다를 수 있음을 확인할 수 있다.
[개념 9] 집단 비교에 대한 옳은 설명 3개를 고르시오.
정답: ②, ③, ⑤ — 집단 비교는 같은 기준과 맥락에서 수행해야 한다.

10 탐색적 분석과 해석의 한계

질문을 만드는 통계

개념 상세 설명: 탐색적 분석은 자료의 특징, 패턴과 이상 현상을 찾아 다음 질문을 만드는 과정이다. 관찰된 차이나 관계를 곧바로 인과관계로 단정하지 않고 표본 크기, 결측과 수집 조건을 함께 고려한다.

summary = {
    "표본수": 30,
    "평균": 23.4,
    "중앙값": 22.8,
    "범위": 18.2,
    "해석": "큰 값 일부가 평균을 높였을 가능성"
}

for key, value in summary.items():
    print(key, ":", value)
코드 상세 설명: 통계값과 해석을 함께 기록한다. 해석은 자료에서 확인한 사실과 가능한 설명을 구분하여 작성해야 한다.
[개념 10] 탐색적 분석에 대한 옳은 설명 3개를 고르시오.
정답: ①, ④, ⑤ — 탐색적 분석은 패턴을 찾고 다음 탐구를 설계하는 과정이다.

📚 학습과 관련한 탐구 보고서 주제 예시 5개

1. 교실 이산화탄소 농도의 시간대별 분포 비교

분석: 시간대별 평균, 중앙값, 범위와 사분위범위를 계산한다.

탐구 관점: 평균이 높은 시간대와 변동이 큰 시간대를 구분하여 환기 시점을 제안한다.

2. 학교 급식 만족도 응답의 빈도와 상대도수 분석

분석: 만족도 범주별 빈도와 학년별 비율을 계산한다.

탐구 관점: 단순 응답 수와 상대도수 비교가 어떤 차이를 만드는지 살펴본다.

3. 이상값이 평균과 중앙값에 미치는 영향 연구

분석: 이상값 포함 전후 평균, 중앙값과 범위를 비교한다.

탐구 관점: 자료 특성에 따라 어떤 대표값이 더 적절한지 논의한다.

4. 두 학급의 학습 시간 분포 비교

분석: 표본 수, 평균, 중앙값, 범위와 사분위범위를 비교한다.

탐구 관점: 중심은 비슷하지만 퍼짐이 다른 경우를 해석한다.
