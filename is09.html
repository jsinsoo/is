6. 학습할 코드 1개

부분 배열을 정렬한 뒤 결합하는 병합 정렬

def merge(left, right):
    merged = []
    i = 0
    j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            merged.append(left[i])
            i += 1
        else:
            merged.append(right[j])
            j += 1

    merged.extend(left[i:])
    merged.extend(right[j:])
    return merged


def merge_sort(data):
    if len(data) <= 1:
        return data

    middle = len(data) // 2
    left = merge_sort(data[:middle])
    right = merge_sort(data[middle:])

    return merge(left, right)


numbers = [8, 3, 6, 2, 7, 1, 5, 4]

print("정렬 전:", numbers)
sorted_numbers = merge_sort(numbers)
print("정렬 후:", sorted_numbers)

상세 설명

코드 요소설명
if len(data) <= 1원소가 0개 또는 1개이면 이미 정렬되어 있으므로 더 이상 분할하지 않는 기저 조건이다.
middle = len(data) // 2배열을 두 부분으로 나눌 가운데 위치를 계산한다.
data[:middle]전체 배열의 왼쪽 부분 문제를 만든다.
data[middle:]전체 배열의 오른쪽 부분 문제를 만든다.
merge_sort(...)왼쪽과 오른쪽 부분 배열을 각각 재귀적으로 정렬한다.
merge(left, right)정렬된 두 부분 배열을 비교하며 하나의 정렬 배열로 결합한다.
merged.extend(...)한쪽 배열의 비교가 끝난 뒤 남아 있는 원소를 결과 뒤에 추가한다.
시간 복잡도배열을 약 log₂n단계로 나누고 각 단계에서 n개의 원소를 병합하므로 일반적으로 O(n log n)이다.

실행 흐름: [8,3,6,2,7,1,5,4]를 절반씩 계속 나눈 뒤, [8]과 [3]을 [3,8]로 합치는 방식으로 작은 정렬 결과를 단계적으로 결합한다.
