7. 학습할 코드 1개

함수·재귀·가지치기·동적계획법을 연결한 종합 코드

search_calls = 0
memo_calls = 0


def find_target_combinations(numbers, target):
    solutions = []

    def search(index, selected, current_sum):
        global search_calls
        search_calls += 1

        if current_sum > target:
            return

        if index == len(numbers):
            if current_sum == target:
                solutions.append(selected[:])
            return

        search(index + 1, selected, current_sum)

        selected.append(numbers[index])
        search(
            index + 1,
            selected,
            current_sum + numbers[index]
        )
        selected.pop()

    search(0, [], 0)
    return solutions


def count_stair_ways(n, memo):
    global memo_calls
    memo_calls += 1

    if n in memo:
        return memo[n]

    memo[n] = (
        count_stair_ways(n - 1, memo)
        + count_stair_ways(n - 2, memo)
    )
    return memo[n]


numbers = [2, 3, 5, 7, 9]
target = 12

combinations = find_target_combinations(numbers, target)

memo = {1: 1, 2: 2}
stairs = 10
ways = count_stair_ways(stairs, memo)

print("목표합 조합:", combinations)
print("탐색 호출 수:", search_calls)
print("계단 방법 수:", ways)
print("메모이제이션 호출 수:", memo_calls)

상세 설명

코드 요소상세 설명
find_target_combinations숫자 목록과 목표값을 매개변수로 받아 목표합을 만드는 모든 조합을 반환하는 함수이다.
내부 함수 search현재 위치, 선택 목록, 현재 합을 상태로 사용하여 재귀적으로 상태 공간을 탐색한다.
selected.append() / pop()현재 숫자를 선택한 상태로 이동한 뒤 재귀 호출이 끝나면 이전 상태로 복원한다. 재귀 호출 스택과 연결된다.
current_sum > target모든 숫자가 양수이므로 목표를 넘은 상태는 해가 될 수 없어 해당 분기를 가지치기한다.
solutions.append(selected[:])정답 상태의 복사본을 저장하여 이후 상태 변경의 영향을 막는다.
count_stair_ways계단 문제를 n-1과 n-2의 부분 문제로 분해하고 결과를 더해 전체 문제를 해결한다.
memo이미 계산한 부분 문제의 결과를 저장해 같은 n을 다시 계산하지 않도록 한다.
search_calls, memo_calls알고리즘이 실제로 몇 번 호출되었는지 기록하여 효율성을 비교하는 근거가 된다.

주의: 목표합 가지치기는 숫자가 모두 양수일 때 안전하다. 음수가 포함되면 현재 합이 목표를 넘더라도 이후 음수를 더해 목표에 도달할 수 있으므로 같은 조건을 그대로 적용하면 안 된다.
