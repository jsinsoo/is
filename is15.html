6. 학습할 코드 1개

evaluation = {
    "정확성": 4,
    "효율성": 3,
    "사용성": 4,
    "안정성": 3,
    "개인정보 보호": 2,
    "공정성": 3,
    "저작권 준수": 5,
    "투명성": 2
}

functional_items = ["정확성", "효율성", "사용성", "안정성"]
ethical_items = ["개인정보 보호", "공정성", "저작권 준수", "투명성"]


def average_score(items):
    total = sum(evaluation[item] for item in items)
    return total / len(items)


def find_improvements(limit=3):
    return [
        item for item, score in evaluation.items()
        if score < limit
    ]


functional_score = average_score(functional_items)
ethical_score = average_score(ethical_items)
improvements = find_improvements()

print("기능 평가 평균:", round(functional_score, 2))
print("윤리 평가 평균:", round(ethical_score, 2))

print("개선 필요 항목")
for item in improvements:
    print("-", item, ":", evaluation[item], "점")
코드 요소상세 설명
evaluation기능적·윤리적 평가 항목과 1~5점의 점수를 저장한 딕셔너리이다.
functional_items정확성, 효율성, 사용성, 안정성처럼 기능 품질과 관련된 항목 목록이다.
ethical_items개인정보 보호, 공정성, 저작권 준수, 투명성 항목 목록이다.
average_score선택된 평가 항목들의 점수를 합해 평균을 계산한다.
find_improvements기준 점수보다 낮은 항목을 찾아 개선 목록으로 반환한다.
limit=3개선 필요 여부를 판단하는 기본 기준값이다.
round(..., 2)평균 점수를 소수 둘째 자리까지 표시한다.

해석: 이 예에서는 개인정보 보호와 투명성이 3점 미만이므로 결과 공유 전에 데이터 비식별화, 수집 동의, 알고리즘 기준과 한계 설명을 보완해야 한다.
