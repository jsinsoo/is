파이썬 완전 문법 300예제

1편 · 기초 문법 1~50예제 — 출력, 변수, 자료형, 입력, 연산자, 형변환

학습 안내

각 예제는 코드 → 실행 결과 → 상세 설명 → 주의점 순서로 구성되어 있습니다. 코드를 직접 실행하면서 값을 바꾸어 보면 문법을 더 빠르게 익힐 수 있습니다.

범위주제
1~10출력과 주석
11~20변수와 기본 자료형
21~30입력과 형변환
31~40산술·비교·논리 연산자
41~50문자열 기초와 문자열 연산

1. 출력과 주석 — 예제 1~10

예제 1. 가장 기본적인 출력

print()기초 
print("안녕하세요!")
실행 결과
안녕하세요!
설명
print() 함수는 화면에 값을 출력합니다. 큰따옴표 또는 작은따옴표로 감싼 내용은 문자열입니다.

예제 2. 숫자 출력

print(100)
print(3.14)
실행 결과
100
3.14
설명
숫자는 따옴표 없이 작성합니다. 따옴표를 붙이면 숫자가 아니라 문자열이 됩니다.

예제 3. 여러 값 한 번에 출력

print("이름:", "홍길동")
print("점수:", 95)
실행 결과
이름: 홍길동
점수: 95
설명
print() 안에 쉼표로 여러 값을 넣으면 기본적으로 한 칸 띄어 출력합니다.

예제 4. sep 옵션 사용

print("2026", "07", "04", sep="-")
실행 결과
2026-07-04
설명
sep은 여러 값 사이에 넣을 구분 문자를 지정합니다. 기본값은 공백입니다.

예제 5. end 옵션 사용

print("파이썬", end=" ")
print("공부하기")
실행 결과
파이썬 공부하기
설명
기본적으로 print()는 출력 후 줄을 바꿉니다. end=" "를 사용하면 줄바꿈 대신 공백을 넣습니다.

예제 6. 줄바꿈 문자

print("첫째 줄\n둘째 줄\n셋째 줄")
실행 결과
첫째 줄
둘째 줄
셋째 줄
설명
\n은 줄바꿈을 뜻하는 이스케이프 문자입니다.

예제 7. 따옴표 출력

print('그는 "파이썬"을 좋아합니다.')
print("그는 '파이썬'을 좋아합니다.")
실행 결과
그는 "파이썬"을 좋아합니다.
그는 '파이썬'을 좋아합니다.
설명
문자열 바깥쪽 따옴표와 안쪽 따옴표를 다르게 사용하면 따옴표를 쉽게 출력할 수 있습니다.

예제 8. 백슬래시 출력

print("C:\\Python\\python.exe")
실행 결과
C:\Python\python.exe
설명
백슬래시 자체를 출력하려면 \\처럼 두 번 작성합니다.

예제 9. 한 줄 주석

# 이 줄은 설명을 위한 주석입니다.
print("주석은 실행되지 않습니다.")
실행 결과
주석은 실행되지 않습니다.
설명
# 뒤에 작성한 내용은 프로그램 실행에 영향을 주지 않습니다. 코드 설명이나 메모에 사용합니다.

예제 10. 여러 줄 문자열

message = """파이썬은
배우기 쉽고
활용 범위가 넓습니다."""
print(message)
실행 결과
파이썬은
배우기 쉽고
활용 범위가 넓습니다.
설명
큰따옴표 3개 또는 작은따옴표 3개를 사용하면 여러 줄 문자열을 만들 수 있습니다.

2. 변수와 기본 자료형 — 예제 11~20

예제 11. 변수에 값 저장하기

name = "홍길동"
print(name)
실행 결과
홍길동
설명
변수는 데이터를 저장하는 이름입니다. =는 오른쪽 값을 왼쪽 변수에 저장한다는 뜻입니다.

예제 12. 변수 값 변경하기

score = 80
print(score)

score = 95
print(score)
실행 결과
80
95
설명
변수에는 새로운 값을 다시 저장할 수 있습니다. 이전 값은 새 값으로 바뀝니다.

예제 13. 여러 변수 만들기

name = "민수"
age = 15
height = 170.5

print(name, age, height)
실행 결과
민수 15 170.5
설명
문자열, 정수, 실수처럼 서로 다른 종류의 데이터를 각각 변수에 저장할 수 있습니다.

예제 14. 여러 변수에 한 번에 값 저장

name, age, city = "서연", 16, "광주"
print(name, age, city)
실행 결과
서연 16 광주
설명
왼쪽 변수 개수와 오른쪽 값 개수가 같으면 한 줄에서 여러 값을 저장할 수 있습니다.

예제 15. 같은 값 여러 변수에 저장

x = y = z = 10
print(x, y, z)
실행 결과
10 10 10
설명
연속 대입을 사용하면 여러 변수에 같은 값을 저장할 수 있습니다.

예제 16. 정수형 int

age = 15
print(age)
print(type(age))
실행 결과
15
<class 'int'>
설명
int는 소수점이 없는 정수를 뜻합니다. type()은 데이터의 자료형을 확인합니다.

예제 17. 실수형 float

pi = 3.14159
print(pi)
print(type(pi))
실행 결과
3.14159
<class 'float'>
설명
float는 소수점이 있는 실수 자료형입니다.

예제 18. 문자열 str

subject = "Python"
print(subject)
print(type(subject))
실행 결과
Python
<class 'str'>
설명
str은 문자와 문장 데이터를 저장하는 문자열 자료형입니다.

예제 19. 불리언 bool

is_student = True
is_raining = False

print(is_student)
print(type(is_raining))
실행 결과
True
<class 'bool'>
설명
bool은 참 또는 거짓을 저장합니다. 반드시 True, False처럼 첫 글자를 대문자로 작성합니다.

예제 20. None 값

result = None
print(result)
print(type(result))
실행 결과
None
<class 'NoneType'>
설명
None은 값이 아직 없음을 나타냅니다. 빈 문자열 ""이나 숫자 0과는 다릅니다.

3. 입력과 형변환 — 예제 21~30

예제 21. input으로 이름 입력받기

name = input("이름을 입력하세요: ")
print("안녕하세요,", name)
입력 예시
이름을 입력하세요: 민수
실행 결과
안녕하세요, 민수
설명
input()은 사용자가 입력한 내용을 문자열로 받아옵니다.

예제 22. 입력값의 자료형 확인

age = input("나이를 입력하세요: ")
print(age)
print(type(age))
설명
사용자가 15를 입력해도 input()의 결과는 "15"라는 문자열입니다.

예제 23. 문자열을 정수로 변환

age = int(input("나이를 입력하세요: "))
print(age + 1)
입력 예시
나이를 입력하세요: 15
실행 결과
16
설명
int()는 숫자 모양의 문자열을 정수로 바꿉니다.

예제 24. 문자열을 실수로 변환

height = float(input("키를 입력하세요: "))
print(height + 0.5)
입력 예시
키를 입력하세요: 170.5
실행 결과
171.0
설명
float()는 소수점이 포함된 문자열을 실수로 변환합니다.

예제 25. 숫자를 문자열로 변환

score = 95
message = "점수는 " + str(score) + "점입니다."
print(message)
실행 결과
점수는 95점입니다.
설명
문자열과 숫자를 +로 연결하려면 숫자를 str()로 문자열로 바꿔야 합니다.

예제 26. bool 형변환

print(bool(1))
print(bool(0))
print(bool("파이썬"))
print(bool(""))
실행 결과
True
False
True
False
설명
0, 빈 문자열, 빈 리스트 등은 보통 False로 변환되고, 값이 있으면 True로 변환됩니다.

예제 27. 두 수의 합

a = int(input("첫 번째 수: "))
b = int(input("두 번째 수: "))

print("합:", a + b)
입력 예시
첫 번째 수: 10
두 번째 수: 20
실행 결과
합: 30
설명
계산을 위해 입력값을 각각 정수로 변환합니다.

예제 28. 문자열 입력을 그대로 연결하기

a = input("첫 번째 수: ")
b = input("두 번째 수: ")

print(a + b)
입력 예시
첫 번째 수: 10
두 번째 수: 20
실행 결과
1020
주의
입력값을 정수로 변환하지 않으면 덧셈이 아니라 문자열 연결이 됩니다.

예제 29. f-string으로 출력하기

name = "지훈"
score = 92

print(f"{name}의 점수는 {score}점입니다.")
실행 결과
지훈의 점수는 92점입니다.
설명
문자열 앞에 f를 붙이고 중괄호 안에 변수를 넣으면 편리하게 값을 출력할 수 있습니다.

예제 30. 입력값으로 자기소개 만들기

name = input("이름: ")
grade = int(input("학년: "))
favorite = input("좋아하는 과목: ")

print(f"{name} 학생은 {grade}학년이고, {favorite} 과목을 좋아합니다.")
설명
문자열 입력, 정수 변환, f-string을 함께 사용하는 기본 예제입니다.

4. 연산자 — 예제 31~40

예제 31. 덧셈과 뺄셈

a = 20
b = 7

print(a + b)
print(a - b)
실행 결과
27
13
설명
+는 덧셈, -는 뺄셈 연산자입니다.

예제 32. 곱셈과 나눗셈

a = 20
b = 6

print(a * b)
print(a / b)
실행 결과
120
3.3333333333333335
설명
*는 곱셈, /는 나눗셈입니다. 나눗셈 결과는 실수형으로 나옵니다.

예제 33. 몫과 나머지

print(17 // 5)
print(17 % 5)
실행 결과
3
2
설명
//는 나눈 몫의 정수 부분, %는 나머지를 구합니다.

예제 34. 거듭제곱

print(2 ** 3)
print(5 ** 2)
실행 결과
8
25
설명
**는 거듭제곱 연산자입니다. 2 ** 3은 2의 3제곱입니다.

예제 35. 복합 대입 연산자

score = 80
score += 10
print(score)

score -= 5
print(score)
실행 결과
90
85
설명
score += 10은 score = score + 10과 같은 뜻입니다.

예제 36. 비교 연산자

print(10 > 5)
print(10 < 5)
print(10 == 10)
print(10 != 5)
실행 결과
True
False
True
True
설명
비교 연산의 결과는 항상 True 또는 False입니다.

예제 37. 크거나 같음, 작거나 같음

score = 80

print(score >= 80)
print(score <= 70)
실행 결과
True
False
설명
>=는 크거나 같음, <=는 작거나 같음을 의미합니다.

예제 38. 논리 and

age = 15
has_ticket = True

print(age >= 14 and has_ticket)
실행 결과
True
설명
and는 두 조건이 모두 참일 때만 True가 됩니다.

예제 39. 논리 or

is_weekend = False
is_holiday = True

print(is_weekend or is_holiday)
실행 결과
True
설명
or는 두 조건 중 하나라도 참이면 True가 됩니다.

예제 40. 논리 not

is_raining = True

print(not is_raining)
실행 결과
False
설명
not은 True와 False를 반대로 바꿉니다.

5. 문자열 기초 — 예제 41~50

예제 41. 문자열 길이

text = "Python"
print(len(text))
실행 결과
6
설명
len()은 문자열에 들어 있는 글자 수를 반환합니다.

예제 42. 문자열 인덱싱

text = "Python"

print(text[0])
print(text[1])
print(text[-1])
실행 결과
P
y
n
설명
문자열의 첫 번째 위치는 0입니다. 음수 인덱스는 뒤에서부터 셉니다.

예제 43. 문자열 슬라이싱

text = "Python Programming"

print(text[0:6])
print(text[7:])
실행 결과
Python
Programming
설명
[시작:끝] 형식으로 문자열 일부를 가져옵니다. 끝 위치의 문자는 포함되지 않습니다.

예제 44. 문자열 연결

first = "파이썬"
second = "프로그래밍"

print(first + " " + second)
실행 결과
파이썬 프로그래밍
설명
문자열끼리는 + 연산자로 연결할 수 있습니다.

예제 45. 문자열 반복

print("★" * 5)
print("안녕! " * 3)
실행 결과
★★★★★
안녕! 안녕! 안녕!
설명
문자열에 정수를 곱하면 해당 횟수만큼 반복됩니다.

예제 46. 대문자와 소문자 변환

text = "Python"

print(text.upper())
print(text.lower())
실행 결과
PYTHON
python
설명
upper()는 대문자, lower()는 소문자로 변환합니다.

예제 47. 공백 제거

text = "   Python Study   "

print(text.strip())
실행 결과
Python Study
설명
strip()은 문자열 양쪽 끝의 공백을 제거합니다.

예제 48. 문자열 치환

text = "나는 자바를 공부합니다."
new_text = text.replace("자바", "파이썬")

print(new_text)
실행 결과
나는 파이썬을 공부합니다.
설명
replace(기존문자열, 새문자열)은 문자열 일부를 바꾼 새 문자열을 반환합니다.

예제 49. 문자열 포함 여부

text = "Python Programming"

print("Python" in text)
print("Java" in text)
실행 결과
True
False
설명
in 연산자는 특정 문자열이 포함되어 있는지 확인합니다.

예제 50. 문자열 포맷 종합

name = "도윤"
subject = "파이썬"
score = 98

message = f"{name} 학생은 {subject} 과목에서 {score}점을 받았습니다."
print(message)
실행 결과
도윤 학생은 파이썬 과목에서 98점을 받았습니다.
설명
f-string은 여러 변수와 문장을 조합해 자연스러운 결과를 만들 때 매우 편리합니다.

1편 핵심 요약

  • print()는 값을 출력하고, input()은 사용자의 입력을 문자열로 받습니다.
  • 기본 자료형에는 int, float, str, bool, None이 있습니다.
  • int(), float(), str()로 자료형을 변환할 수 있습니다.
  • 산술 연산자, 비교 연산자, 논리 연산자를 사용해 계산과 조건 판단을 수행합니다.
  • 문자열은 인덱싱, 슬라이싱, 연결, 반복, 메서드 호출로 다양하게 다룰 수 있습니다.
