성취기준

영역수업에 적용한 성취기준
유형 선택자료 유형과 전달 목적에 따라 막대그래프, 선그래프, 산점도와 히스토그램을 적절히 선택한다.
표현 설계제목, 축 이름, 단위, 범례, 색상과 주석을 활용하여 그래프를 명확하게 구성한다.
왜곡 방지축 범위, 비율, 구간 폭과 생략된 정보가 해석에 미치는 영향을 점검한다.
정보 전달시각화가 보여 주는 사실과 보여 주지 못하는 한계를 구분하여 설명한다.

학습 목표

  1. 범주 비교, 시간 변화, 변수 관계와 분포에 적합한 시각화 유형을 선택할 수 있다.
  2. 막대그래프, 선그래프, 산점도와 히스토그램을 파이썬으로 구현할 수 있다.
  3. 그래프의 제목, 축 이름, 단위와 범례를 정확하게 설정할 수 있다.
  4. 색상과 주석을 필요한 정보 강조에 절제하여 사용할 수 있다.
  5. 축 절단, 불균일한 구간과 과도한 장식이 일으키는 오해를 설명할 수 있다.
  6. 시각화의 전달 효과와 한계를 점검하고 개선안을 제시할 수 있다.

1 전달 목적과 시각화 유형 선택

먼저 질문을 정한다

개념 상세 설명: 좋은 시각화는 그래프 종류보다 전달 목적에서 시작한다. 범주별 크기 비교에는 막대그래프, 시간 변화에는 선그래프, 두 수치 변수의 관계에는 산점도, 연속형 값의 분포에는 히스토그램이 적합하다.

purpose_to_chart = {
    "범주 비교": "막대그래프",
    "시간 변화": "선그래프",
    "두 변수 관계": "산점도",
    "연속형 분포": "히스토그램"
}

purpose = "시간 변화"
print("추천 유형:", purpose_to_chart[purpose])
코드 상세 설명: 전달 목적을 키로 하고 적합한 그래프 유형을 값으로 저장한다. 시각화 전에 “무엇을 보여 줄 것인가”를 먼저 정하는 습관을 기른다.
[개념 1] 시각화 유형 선택에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ⑤ — 자료의 성격과 전달 목적에 맞춰 유형을 선택한다.

2 막대그래프

범주별 크기 비교

개념 상세 설명: 막대그래프는 서로 독립적인 범주의 크기나 빈도를 비교하는 데 적합하다. 막대 사이에는 간격이 있으며, 값의 크기를 막대 길이로 나타낸다.

import matplotlib.pyplot as plt

categories = ["물리", "화학", "생물", "지구과학"]
counts = [18, 15, 21, 12]

plt.bar(categories, counts)
plt.title("과학 분야별 탐구 주제 수")
plt.xlabel("분야")
plt.ylabel("주제 수(개)")
plt.show()
코드 상세 설명: plt.bar()로 범주별 값을 비교한다. 제목과 축 이름에 대상과 단위를 명확히 적어 해석을 돕는다.
[개념 2] 막대그래프에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ④ — 막대그래프는 독립 범주의 크기를 길이로 비교한다.

3 선그래프

시간과 순서의 변화

개념 상세 설명: 선그래프는 시간이나 순서가 있는 값의 변화를 나타낸다. 점을 선으로 연결하므로 증가·감소·변곡과 추세를 파악하기 쉽다. 순서가 없는 범주를 임의로 연결하면 잘못된 연속성을 만들 수 있다.

import matplotlib.pyplot as plt

hours = [9, 10, 11, 12, 13, 14]
co2 = [620, 710, 840, 980, 760, 690]

plt.plot(hours, co2, marker="o")
plt.title("시간에 따른 교실 이산화탄소 농도")
plt.xlabel("시각")
plt.ylabel("이산화탄소 농도(ppm)")
plt.grid(True, alpha=0.3)
plt.show()
코드 상세 설명: 시간 순서에 따라 값을 선으로 연결한다. 표식은 관측 지점을 보여 주고 격자는 값을 읽는 데 도움을 준다.
[개념 3] 선그래프에 대한 옳은 설명 3개를 고르시오.
정답: ②, ③, ⑤ — 선그래프는 순서와 변화가 핵심일 때 사용한다.

4 산점도

두 수치 변수의 관계

개념 상세 설명: 산점도는 각 관측값을 x축과 y축의 좌표점으로 나타내 두 수치 변수의 관계, 군집, 이상값과 방향성을 탐색한다. 관계가 보여도 곧바로 인과관계로 해석해서는 안 된다.

import matplotlib.pyplot as plt

temperature = [18, 20, 22, 24, 26, 28]
electricity = [120, 125, 132, 148, 170, 198]

plt.scatter(temperature, electricity)
plt.title("기온과 전력 사용량의 관계")
plt.xlabel("기온(℃)")
plt.ylabel("전력 사용량(kWh)")
plt.show()
코드 상세 설명: 각 기온과 전력 사용량 쌍을 하나의 점으로 표현한다. 점들의 방향과 퍼짐을 살펴 관계의 가능성을 탐색한다.
[개념 4] 산점도에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ④ — 산점도는 관계를 탐색하지만 인과를 증명하지 않는다.

5 히스토그램

연속형 자료의 분포

개념 상세 설명: 히스토그램은 연속형 수치 자료를 구간으로 나누고 각 구간의 빈도를 막대로 나타낸다. 막대가 붙어 있으며, 구간 수와 폭에 따라 분포의 인상이 달라질 수 있다.

import matplotlib.pyplot as plt

scores = [55, 62, 67, 68, 71, 73, 75, 78, 82, 86, 90, 94]

plt.hist(scores, bins=5, edgecolor="black")
plt.title("점수 분포")
plt.xlabel("점수 구간")
plt.ylabel("빈도")
plt.show()
코드 상세 설명: bins=5는 자료 범위를 5개 구간으로 나눈다. 구간 수를 바꾸며 분포 특징이 안정적으로 나타나는지 확인해야 한다.
[개념 5] 히스토그램에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ⑤ — 히스토그램은 구간 설정이 해석에 중요하다.

6 축의 범위와 눈금

과장과 축소를 막는다

개념 상세 설명: 축은 값의 위치와 크기를 해석하는 기준이다. 막대그래프의 수직축을 중간값에서 시작하면 작은 차이가 크게 보일 수 있다. 선그래프에서는 변화가 잘 보이도록 범위를 조정할 수 있지만 축 절단 사실을 명확히 알려야 한다.

import matplotlib.pyplot as plt

classes = ["A", "B", "C"]
scores = [82, 84, 86]

plt.bar(classes, scores)
plt.ylim(0, 100)
plt.title("학급별 평균 점수")
plt.ylabel("평균 점수(점)")
plt.show()
코드 상세 설명: y축을 0에서 100까지 설정해 막대 길이가 실제 점수 비율과 지나치게 다르게 보이지 않도록 한다.
[개념 6] 축 설계에 대한 옳은 설명 3개를 고르시오.
정답: ②, ④, ⑤ — 축은 단위, 범위와 간격을 투명하게 제시해야 한다.

7 제목·축 이름·단위·출처

그래프의 문맥

개념 상세 설명: 제목은 무엇을 언제 어디서 측정했는지 드러내고, 축 이름은 변수와 단위를 알려야 한다. 필요한 경우 자료 기간, 표본 수와 출처를 덧붙이면 그래프를 독립적으로 이해할 수 있다.

import matplotlib.pyplot as plt

months = ["3월", "4월", "5월", "6월"]
rain = [42, 61, 88, 121]

plt.plot(months, rain, marker="o", label="월 강수량")
plt.title("2026년 3~6월 지역 월 강수량")
plt.xlabel("월")
plt.ylabel("강수량(mm)")
plt.legend()
plt.figtext(0.5, 0.01, "자료: 수업용 예시 자료", ha="center")
plt.show()
코드 상세 설명: 제목, 축 이름, 단위, 범례와 자료 출처를 함께 표시한다. 그래프만 보아도 대상과 측정값을 이해할 수 있도록 구성한다.
[개념 7] 그래프의 문맥 정보에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ⑤ — 제목, 단위와 출처는 그래프의 의미를 분명하게 한다.

8 범례와 색상 설계

구분은 명확하게, 장식은 절제

개념 상세 설명: 여러 자료 계열이 있을 때 범례는 선, 막대와 점이 무엇을 뜻하는지 알려 준다. 색상은 중요한 차이를 구분하는 데 사용하되 너무 많은 색을 쓰지 않고, 색각 다양성을 고려해 선 모양이나 표식을 함께 활용한다.

import matplotlib.pyplot as plt

months = ["3월", "4월", "5월", "6월"]
class_a = [68, 72, 77, 81]
class_b = [65, 70, 74, 80]

plt.plot(months, class_a, marker="o", label="A학급")
plt.plot(months, class_b, marker="s", linestyle="--", label="B학급")
plt.title("학급별 월간 평균 점수")
plt.ylabel("평균 점수(점)")
plt.legend()
plt.show()
코드 상세 설명: 색상뿐 아니라 표식과 선 모양을 다르게 하여 흑백 인쇄나 색각 차이에서도 두 계열을 구분할 수 있도록 한다.
[개념 8] 범례와 색상 설계에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ④ — 색상과 범례는 명확하고 접근 가능하게 설계한다.

9 주석과 핵심 정보 강조

필요한 곳만 설명한다

개념 상세 설명: 주석은 최고점, 급격한 변화, 정책 시행 시점이나 이상값처럼 해석에 중요한 지점을 설명한다. 모든 점에 설명을 붙이면 오히려 읽기 어려우므로 전달 목적에 필요한 정보만 강조한다.

import matplotlib.pyplot as plt

days = [1, 2, 3, 4, 5]
visitors = [120, 135, 128, 210, 175]

plt.plot(days, visitors, marker="o")
plt.annotate(
    "과학 행사 개최",
    xy=(4, 210),
    xytext=(2.8, 225),
    arrowprops={"arrowstyle": "->"}
)
plt.title("일자별 전시관 방문자 수")
plt.xlabel("일")
plt.ylabel("방문자 수(명)")
plt.show()
코드 상세 설명: 방문자 수가 급증한 4일째에 행사 정보를 주석으로 표시한다. 변화의 맥락을 전달하지만 원인을 확정하는 표현은 피한다.
[개념 9] 주석과 강조에 대한 옳은 설명 3개를 고르시오.
정답: ②, ③, ⑤ — 주석은 핵심 맥락을 간결하게 설명해야 한다.

10 시각화 왜곡 방지와 검토

사실을 정직하게 보여 준다

개념 상세 설명: 3차원 효과, 면적과 부피를 이용한 과장, 불균일한 시간 간격, 선택적 자료 제외와 잘린 축은 오해를 만들 수 있다. 시각화 전후에 자료 누락, 축, 단위, 범례와 설명의 정확성을 점검해야 한다.

visual_checklist = {
    "자료 누락 없음": True,
    "축 이름과 단위 표시": True,
    "눈금 간격 일정": True,
    "범례 의미 명확": True,
    "과도한 3차원 효과 없음": True,
    "출처와 기간 표시": True
}

for item, passed in visual_checklist.items():
    print(item, ":", "통과" if passed else "수정 필요")
코드 상세 설명: 시각화 점검 항목을 딕셔너리로 관리한다. 하나라도 통과하지 못하면 그래프를 수정하고 다시 검토한다.
[개념 10] 시각화 왜곡 방지에 대한 옳은 설명 3개를 고르시오.
정답: ①, ④, ⑤ — 정확성과 투명성이 장식보다 우선한다.

📚 학습과 관련한 탐구 보고서 주제 예시 5개

1. 축의 시작점이 학급 평균 비교 인식에 미치는 영향

분석: 동일 자료를 y축 0 시작 그래프와 축 절단 그래프로 제작한다.

탐구 관점: 차이의 크기에 대한 독자의 판단이 어떻게 달라지는지 조사한다.

2. 막대그래프와 선그래프의 정보 전달 효과 비교

분석: 범주 자료와 시간 자료를 두 유형으로 각각 표현한다.

탐구 관점: 그래프 유형과 자료 성격이 맞지 않을 때 생기는 오해를 분석한다.

3. 히스토그램 구간 수에 따른 분포 해석 변화

분석: 같은 점수 자료에 서로 다른 구간 수를 적용한다.

탐구 관점: 봉우리, 치우침과 이상값이 어떻게 다르게 보이는지 비교한다.

4. 색상과 범례 설계가 그래프 이해도에 미치는 영향

분석: 색상만 다른 그래프와 표식·선 모양을 함께 사용한 그래프를 비교한다.

탐구 관점: 색각 다양성과 흑백 출력에서의 구분 가능성을 평가한다.

5. 산점도 표현을 활용한 환경 변수 관계 탐색

분석: 기온과 전력 사용량 또는 미세먼지와 습도의 관계를 산점도로 나타낸다.

탐구 관점: 관계의 방향, 군집과 이상값을 해석하고 인과 단정의 한계를 제시한다.
