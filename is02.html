2. 학습할 코드

자연수 n까지의 합을 재귀적으로 계산합니다. 입력 검증, 종료 조건, 재귀 호출, 반환 과정이 모두 드러나는 예제입니다.

recursive_sum.py
def sum_to(n):
    """1부터 n까지의 자연수 합을 재귀적으로 계산한다."""
    if not isinstance(n, int) or n < 1:
        raise ValueError("n은 1 이상의 정수여야 합니다.")

    if n == 1:              # 종료 조건
        return 1

    return n + sum_to(n - 1)  # 재귀 호출


number = 5
result = sum_to(number)
print(f"1부터 {number}까지의 합: {result}")
코드 요소상세 설명개념적 의미
sum_to(n)자연수 n을 입력받아 1부터 n까지의 합을 반환하는 함수다.문제를 입력 크기 n으로 일반화한다.
isinstance(n, int)n이 정수인지 확인하고 1 미만의 값도 거부한다.재귀가 예상하지 못한 입력으로 무한 진행되는 것을 예방한다.
if n == 1더 이상 작은 문제로 나누지 않고 직접 1을 반환한다.종료 조건은 재귀의 출발점이자 안전장치다.
n + sum_to(n-1)현재 수 n과 더 작은 문제의 답을 결합한다.S(n)=n+S(n-1)이라는 재귀관계를 코드로 표현한다.
sum_to(5)5→4→3→2→1 순으로 호출한 뒤 1→3→6→10→15 순으로 반환된다.호출은 내려가고 결과는 역순으로 올라오는 호출 스택을 보여 준다.
예상 실행 결과
1부터 5까지의 합: 15

수업 끝 멘트

시는 한 행이 다음 행을 부르고, 소설은 한 장면이 다음 장면의 이유가 되며, 희곡과 연극은 앞선 대사가 뒤의 행동을 깨웁니다. 음악도 주제를 되풀이하며 조금씩 더 큰 울림을 만듭니다. 재귀함수 역시 작은 답이 돌아와 큰 답을 완성합니다. 오늘 막혔다면 걱정하지 마세요. 주인공도 첫 장면부터 결말을 알지는 못합니다. 종료 조건만 잊지 않으면 여러분의 꿈도 무한 반복이 아니라 멋진 완결을 향해 갑니다.
