6. 학습할 코드 1개

실생활 문제 후보의 우선순위를 계산하는 프로그램

problems = [
    {
        "name": "급식 대기시간 증가",
        "impact": 5,
        "data": 4,
        "automation": 4,
        "feasibility": 5
    },
    {
        "name": "교실 공기질 악화",
        "impact": 5,
        "data": 5,
        "automation": 4,
        "feasibility": 4
    },
    {
        "name": "도서관 빈자리 확인 어려움",
        "impact": 3,
        "data": 3,
        "automation": 5,
        "feasibility": 4
    }
]


def calculate_score(problem):
    return (
        problem["impact"] * 0.35
        + problem["data"] * 0.25
        + problem["automation"] * 0.20
        + problem["feasibility"] * 0.20
    )


for problem in problems:
    problem["score"] = calculate_score(problem)

ranked = sorted(
    problems,
    key=lambda item: item["score"],
    reverse=True
)

print("문제 우선순위")
for rank, problem in enumerate(ranked, start=1):
    print(
        rank,
        problem["name"],
        round(problem["score"], 2)
    )

상세 설명

코드 요소설명
problems발견한 문제 후보를 딕셔너리로 저장한 리스트이다.
impact문제가 사용자와 학교생활에 미치는 영향 정도를 1~5점으로 나타낸다.
data문제 분석에 필요한 데이터를 실제로 수집할 수 있는 정도이다.
automation알고리즘이나 프로그램으로 반복 작업을 줄이거나 판단을 지원할 수 있는 정도이다.
feasibility시간, 기술, 비용, 개인정보 보호 조건을 고려한 실제 구현 가능성이다.
calculate_score네 평가 기준에 가중치를 곱해 문제의 우선순위 점수를 계산한다.
sorted(..., reverse=True)점수가 높은 문제부터 내림차순으로 정렬한다.
enumerate(..., start=1)정렬된 문제에 1위부터 순위를 붙인다.

중요: 점수가 가장 높다고 반드시 최종 주제가 되는 것은 아니다. 개인정보 침해 가능성, 편향, 안전성, 실제 사용자 의견도 함께 검토해야 한다.
