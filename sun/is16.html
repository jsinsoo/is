Ⅱ. 학습 목표와 학습할 개념

학습 목표

  1. 문제의 상태·연산·목표 상태를 정의한다.
  2. 중첩 반복문으로 모든 후보를 생성한다.
  3. 순열·조합·부분집합을 백트래킹으로 구현한다.
  4. 가지치기와 상태 복원의 역할을 설명한다.
  5. 후보 수와 시간·메모리 효율을 분석한다.

학습할 개념

  • 상태 공간: 문제 해결 과정에서 가능한 모든 상태의 집합
  • 완전 탐색: 가능한 후보를 모두 생성하고 검사하는 방법
  • 백트래킹: 선택 후 탐색하고 상태를 복원하는 방법
  • 가지치기: 해가 될 수 없는 후보를 조기에 중단하는 전략
  • 복원: 선택 목록·방문 정보·누적값을 이전 상태로 되돌리는 과정
개념 일반화
완전 탐색은 후보를 빠짐없이 확인하여 정답을 보장하고, 백트래킹은 상태를 선택·탐색·복원하며 가지치기를 통해 불가능한 후보를 줄인다.

Ⅲ. 수업 시작 멘트

“소크라테스는 가능한 질문을 하나씩 따라가며 생각의 빈틈을 줄였고, 데카르트는 복잡한 문제를 작은 상태로 나누었습니다. 다윈은 수많은 변이 가운데 살아남는 선택을 관찰했고, 파스칼은 선택의 경우가 어떻게 조합의 삼각형을 만드는지 보여 주었습니다. 오늘 우리는 모든 가능성을 탐색하지만 아무 생각 없이 헤매지는 않습니다. 목표가 아닌 길은 가지치기하고, 잘못 선택한 길은 백트래킹으로 돌아옵니다. 단, 시험에서 모든 답을 완전 탐색하면 감독 선생님이 가지치기를 실행할 수 있으니 오늘 코드에서 충분히 연습해 봅시다.”

def generate_combinations(
    items, r, start=0, selected=None
):
    if selected is None:
        selected = []

    if len(selected) == r:
        print(selected)
        return

    for index in range(start, len(items)):
        selected.append(items[index])

        generate_combinations(
            items,
            r,
            index + 1,
            selected
        )

        selected.pop()

generate_combinations(

def generate_subsets(
    items, index=0, selected=None
):
    if selected is None:
        selected = []

    if index == len(items):
        print(selected)
        return

    # 현재 원소 포함

def search(start, total, selected):
    if total == target:
        solutions.append(selected.copy())
        return

        return

        coin = coins[index]

valid_arrangements = []

def arrange(selected):
    if len(selected) == len(students):
        # 민준과 서연이 붙어 앉지 않는 조건

            arrange(selected)
            selected.pop()

예제 9

미로에서 모든 가능한 경로 탐색

상태 공간방문 복원
maze = [
    [0, 0, 0],
    [0, 1, 0],
    [0, 0, 0]
]

rows = len(maze)
cols = len(maze[0])
target = (2, 2)

visited = set()
paths = []

def explore(row, col, path):
    if not (0 <= row < rows and 0 <= col < cols):
        return

    if maze[row][col] == 1:
        return

    if (row, col) in visited:
        return

    visited.add((row, col))
    path.append((row, col))

    if (row, col) == target:
        paths.append(path.copy())
    else:
        for dr, dc in [
            (1, 0), (-1, 0),
            (0, 1), (0, -1)
        ]:
            explore(
                row + dr,
                col + dc,
                path
            )

    path.pop()
    visited.remove((row, col))

explore(0, 0, [])

print("경로 수:", len(paths))
for path in paths:
    print(path)
상태 복원
다른 경로에서 같은 칸을 다시 사용할 수 있도록 재귀 호출이 끝난 뒤 방문 표시와 경로를 원상 복구합니다.

예제 10

종합: 목표 합 부분집합 백트래킹

종합 설계가지치기해 기록
numbers = [2, 3, 5, 7, 8]
target = 10
solutions = []

def subset_sum(
    index,
    total,
    selected
):
    # 목표 상태
    if total == target:
        solutions.append(selected.copy())
        return

    # 실패 상태와 가지치기
    if total > target:
        return

    if index == len(numbers):
        return

    # 현재 숫자를 선택
    selected.append(numbers[index])
    subset_sum(
        index + 1,
        total + numbers[index],
        selected
    )

    # 선택 복원
    selected.pop()

    # 현재 숫자를 선택하지 않음
    subset_sum(
        index + 1,
        total,
        selected
    )

subset_sum(0, 0, [])

print("목표 합:", target)
print("해의 개수:", len(solutions))

for solution in solutions:
    print(solution)

연산현재 숫자를 선택하거나 선택하지 않음
목표 상태누적 합이 target과 같음
실패 상태합이 목표를 초과하거나 모든 숫자를 검사함
복원선택한 숫자를 pop하여 이전 상태로 돌아감
