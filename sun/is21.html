학습할 개념

  • 분할: 문제를 더 작은 부분 문제로 나누는 과정
  • 해결: 부분 문제를 재귀적으로 처리하는 과정
  • 결합: 부분 결과를 전체 답으로 합치는 과정

  • 복잡도: 병합 정렬 O(n log n), 이진 탐색 O(log n)

예제 1

정렬된 두 리스트 병합하기

결합 단계투 포인터
def merge(left, right):
    result = []
    i = 0
    j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result.extend(left[i:])
    result.extend(right[j:])

    return result

print(merge(
    [1, 4, 7],
    [2, 3, 8]
))

결합 과정

[1,4,7] + [2,3,8]

1과 2 비교 → 1
4와 2 비교 → 2
4와 3 비교 → 3
4와 8 비교 → 4
7과 8 비교 → 7
남은 8 추가

결과: [1,2,3,4,7,8]

예제 2

기본 병합 정렬 구현

분할재귀 해결병합
def merge_sort(data):
    # 종료 조건
    if len(data) <= 1:
        return data

    # 1. 분할
    middle = len(data) // 2
    left_data = data[:middle]
    right_data = data[middle:]

    # 2. 해결
    left_sorted = merge_sort(left_data)
    right_sorted = merge_sort(right_data)

    # 3. 결합
    return merge(left_sorted, right_sorted)

numbers = [7, 3, 8, 2, 5, 1]
print(merge_sort(numbers))
구조
배열 길이가 1 이하가 될 때까지 절반으로 나눈 뒤, 정렬된 작은 배열부터 순서대로 병합합니다.

def merge_sort_trace(data, depth=0):
    indent = "  " * depth
    print(f"{indent}분할 대상: {data}")

예제 4

병합 단계의 비교 횟수 측정

비교 횟수성능 분석
comparison_count = 0

def merge_count(left, right):
    global comparison_count

    result = []
    i = j = 0

    while i < len(left) and j < len(right):
        comparison_count += 1

        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result.extend(left[i:])
    result.extend(right[j:])

    return result

def merge_sort_count(data):
    if len(data) <= 1:
        return data

    middle = len(data) // 2

    left = merge_sort_count(data[:middle])
    right = merge_sort_count(data[middle:])

    return merge_count(left, right)

numbers = [8, 7, 6, 5, 4, 3, 2, 1]

result = merge_sort_count(numbers)

print("정렬 결과:", result)
print("비교 횟수:", comparison_count)
시간 관계
각 단계의 병합은 전체 원소 수에 비례하고, 분할 깊이는 대략 log₂n 단계이므로 전체 시간은 일반적으로 O(n log n)입니다.

예제 5

인덱스 범위를 이용한 병합 정렬

제자리 수정보조 배열
def merge_range(data, left, middle, right):
    temp = []
    i = left
    j = middle + 1

    while i <= middle and j <= right:
        if data[i] <= data[j]:
            temp.append(data[i])
            i += 1
        else:
            temp.append(data[j])
            j += 1

    temp.extend(data[i:middle + 1])
    temp.extend(data[j:right + 1])

    data[left:right + 1] = temp

def merge_sort_range(data, left, right):
    if left >= right:
        return

    middle = (left + right) // 2

    merge_sort_range(data, left, middle)
    merge_sort_range(data, middle + 1, right)

    merge_range(data, left, middle, right)

numbers = [9, 1, 6, 3, 8, 2]
merge_sort_range(numbers, 0, len(numbers) - 1)

print(numbers)
공간 관점
원본 리스트를 직접 수정하지만 병합을 위해 보조 리스트가 필요합니다. 병합 정렬은 일반적으로 O(n)의 추가 공간을 사용합니다.

예제 6

반복문으로 이진 탐색 구현

탐색 범위 절반반복 방식
def binary_search(data, target):
    left = 0
    right = len(data) - 1

    while left <= right:
        middle = (left + right) // 2

        if data[middle] == target:
            return middle

        if data[middle] < target:
            left = middle + 1
        else:
            right = middle - 1

    return -1

numbers = [2, 5, 8, 12, 16, 23, 38]
print(binary_search(numbers, 16))

목표가 크면 오른쪽 절반, 작으면 왼쪽 절반만 남깁니다.

예제 7

재귀함수로 이진 탐색 구현

재귀 탐색종료 조건
def binary_search_recursive(
    data,
    target,
    left,
    right
):
    if left > right:
        return -1

    middle = (left + right) // 2

    if data[middle] == target:
        return middle

    if data[middle] < target:
        return binary_search_recursive(
            data,
            target,
            middle + 1,
            right
        )

    return binary_search_recursive(
        data,
        target,
        left,
        middle - 1
    )

numbers = [2, 5, 8, 12, 16, 23, 38]

print(binary_search_recursive(
    numbers,
    23,
    0,
    len(numbers) - 1
))
종료 조건
left > right이면 탐색 범위가 사라진 것이므로 목표값이 없다고 판단합니다.

def binary_search_trace(data, target):

def find_first(data, target):
    left = 0
    right = len(data) - 1
    answer = -1

    while left <= right:
        middle = (left + right) // 2

        if data[middle] == target:
            answer = middle
            right = middle - 1

        elif data[middle] < target:
            left = middle + 1

        else:
            right = middle - 1

    return answer

numbers = [1, 2, 2, 2, 3, 4, 5]

Ⅶ. 수업 끝 멘트

시는 긴 감정을 짧은 행으로 나누어 마음에 남기고, 소설은 수많은 장면을 한 이야기로 결합합니다. 희곡은 막과 장으로 나뉘지만 마지막에는 하나의 메시지가 되고, 연극은 배우·조명·음향의 작은 역할이 모여 큰 무대를 완성합니다. 음악도 한 음 한 음은 작지만 리듬과 화음 속에서 거대한 울림이 됩니다.

여러분의 꿈도 한 번에 완성할 수 없는 큰 문제일 수 있습니다. 그러나 오늘 할 일, 이번 주의 목표, 다음 단계의 도전으로 나누고 하나씩 해결하면 언젠가 멋진 결과로 결합됩니다.
