예제 4

팩토리얼 호출과 복귀 과정 추적

호출 스택복귀 추적
def factorial_trace(n, depth=0):
    indent = "  " * depth
    print(f"{indent}호출: factorial({n})")

    if n == 0:
        print(f"{indent}반환: 1")
        return 1

    result = n * factorial_trace(n - 1, depth + 1)
    print(f"{indent}반환: {result}")
    return result

print("최종 결과:", factorial_trace(4))

호출 단계

factorial(4)
  factorial(3)
    factorial(2)
      factorial(1)
        factorial(0)

복귀 단계

factorial(0) → 1
factorial(1) → 1×1 = 1
factorial(2) → 2×1 = 2
factorial(3) → 3×2 = 6
factorial(4) → 4×6 = 24
핵심
호출은 4→3→2→1→0 순서로 진행되고, 반환은 0→1→2→3→4의 역순으로 진행됩니다.

def gcd_trace(a, b, depth=0):

“시는 한 줄의 이미지가 다음 줄의 마음을 호출하고, 소설은 작은 사건들이 쌓여 한 사람의 삶을 완성합니다. 희곡은 장면이 바뀔 때마다 인물이 다시 등장하고, 연극은 막이 내려가야 다음 막이 시작되며, 음악은 되풀이되는 선율 속에서도 마지막 음을 향해 나아갑니다. 여러분의 꿈도 오늘 한 번의 호출로 완성되지는 않습니다. 작은 도전이 다음 도전을 부르고, 실패에서 돌아온 값이 다음 선택의 매개변수가 됩니다. 다만 밤새 공부를 무한 재귀로 돌리면 선생님보다 먼저 RecursionError가 날 수 있으니 휴식이라는 종료 조건도 잊지 마세요. 오늘의 작은 성공을 반환값으로 품고, 내일의 더 큰 함수를 힘차게 호출합시다.”
