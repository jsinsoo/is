학습할 개념

  • 상태 복원: 방문·선택·충돌 표시를 이전 상태로 되돌리는 과정
  • 중복 방지: 같은 결과를 여러 번 생성하지 않도록 후보를 제한하는 규칙

개념 일반화
백트래킹은 공통적으로 후보를 선택하고 다음 상태를 재귀 탐색한 뒤 상태를 복원하지만, 효율적인 설계는 문제별 중복과 충돌 조건을 재귀 호출 전에 제거한다.

Ⅲ. 수업 시작 멘트

“소크라테스는 하나의 답에 머물지 않고 다른 가능성을 다시 물었고, 데카르트는 복잡한 문제를 작은 선택으로 나누었습니다. 다윈은 수많은 변이 중 환경에 맞는 선택이 살아남는 과정을 관찰했고, 가우스는 규칙을 발견해 불필요한 계산을 줄였습니다. 오늘 우리는 후보를 하나 고르고, 더 깊이 탐색하고, 맞지 않으면 되돌립니다. 인생에서 선택을 되돌리는 버튼은 드물지만 코드에서는 pop()이 친절하게 기다리고 있습니다. 다만 시험 답안을 제출한 뒤에는 backtracking이 지원되지 않을 수 있으니 오늘 충분히 연습해 봅시다.”

예제 1

선택·재귀·되돌리기의 최소 구조

백트래킹 틀append·pop
items = ["A", "B"]
selected = []

def search(depth):
    if depth == len(items):
        print("완성:", selected)
        return

    for item in items:
        if item in selected:
            continue

        # 1. 선택
        selected.append(item)
        print("선택:", selected)

        # 2. 재귀
        search(depth + 1)

        # 3. 되돌리기
        removed = selected.pop()
        print("복원:", removed, selected)

search(0)

핵심 코드

selected.append(item)  # 선택
search(depth + 1)      # 재귀
selected.pop()         # 되돌리기

예제 4

중복 원소가 있는 순열의 중복 방지

중복 순열같은 깊이 집합
items = ["A", "A", "B"]
selected = []
used = [False] * len(items)
results = []

def unique_permutation():
    if len(selected) == len(items):
        results.append(selected.copy())
        return

    used_this_depth = set()

    for index, item in enumerate(items):
        if used[index]:
            continue

        if item in used_this_depth:
            continue

        used_this_depth.add(item)
        used[index] = True
        selected.append(item)

        unique_permutation()

        selected.pop()
        used[index] = False

unique_permutation()

for result in results:
    print(result)
중복 방지 원리
같은 재귀 깊이에서 동일한 값은 한 번만 선택하여 결과가 같은 순열을 반복 생성하지 않습니다.

        solutions.append(selected.copy())

예제 7

N-Queen의 안전한 열 검사

N-Queen충돌 검사
n = 4

# queens[row] = col
queens = [-1] * n

def is_safe(row, col):
    for previous_row in range(row):
        previous_col = queens[previous_row]

        # 같은 열
        if previous_col == col:
            return False

        # 같은 대각선
        if abs(row - previous_row) == abs(
            col - previous_col
        ):
            return False

    return True

queens[0] = 1

for col in range(n):
    print(
        "1행", col, "열:",
        is_safe(1, col)
    )

상태 표현

queens[row] = col은 해당 행의 퀸 열 위치를 의미합니다.

행마다 퀸을 하나씩 놓으므로 행 충돌은 자동으로 방지됩니다.

예제 8

4-Queen의 모든 해 출력

4-Queen보드 출력
n = 4
queens = [-1] * n
solutions = []

def is_safe(row, col):
    for previous_row in range(row):
        previous_col = queens[previous_row]

        if previous_col == col:
            return False

        if abs(row - previous_row) == abs(
            col - previous_col
        ):
            return False

    return True

def solve(row):
    if row == n:
        solutions.append(queens.copy())
        return

    for col in range(n):
        if not is_safe(row, col):
            continue

        # 선택
        queens[row] = col

        # 재귀
        solve(row + 1)

        # 되돌리기
        queens[row] = -1

solve(0)

for solution in solutions:
    print("해:", solution)

    for row in range(n):
        line = [
            "Q" if solution[row] == col else "."
            for col in range(n)
        ]
        print(" ".join(line))

    print()
목표 상태
row == n이면 모든 행에 충돌 없이 퀸을 배치한 것이므로 하나의 해를 기록합니다.

예제 9

열·대각선 집합을 이용한 N-Queen 최적화

집합빠른 충돌 확인
def count_n_queens(n):
    columns = set()
    diagonal1 = set()  # row - col
    diagonal2 = set()  # row + col
    count = 0

    def solve(row):
        nonlocal count

        if row == n:
            count += 1
            return

        for col in range(n):
            d1 = row - col
            d2 = row + col

            if (
                col in columns
                or d1 in diagonal1
                or d2 in diagonal2
            ):
                continue

            # 선택
            columns.add(col)
            diagonal1.add(d1)
            diagonal2.add(d2)

            # 재귀
            solve(row + 1)

            # 되돌리기
            columns.remove(col)
            diagonal1.remove(d1)
            diagonal2.remove(d2)

    solve(0)
    return count

for n in range(1, 9):
    print(n, "Queen 해의 수:", count_n_queens(n))
효율성
이전 모든 퀸을 반복해서 확인하는 대신, 사용 중인 열과 두 대각선을 집합에서 빠르게 검사합니다.
