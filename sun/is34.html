        book

예제 4

교내 최단 경로 안내 발표

그래프BFS
from collections import deque

def shortest_path(graph, start, target):
    queue = deque([(start, [start])])
    visited = {start}

    while queue:
        node, path = queue.popleft()

        if node == target:
            return path

        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(
                    (neighbor, path + [neighbor])
                )

    return None
발표 항목설명
설계장소를 노드, 통로를 간선으로 표현함.
알고리즘모든 이동 비용이 같아 BFS를 선택함.
검증출발지와 도착지가 같을 때, 경로가 없을 때를 테스트함.
성찰문제를 그래프로 바꾸면 실제 공간 문제를 알고리즘으로 해결할 수 있음을 이해함.

jobs = deque([

Ⅶ. 수업 끝 멘트

시는 마지막 행에서 처음의 감정을 새롭게 비추고, 소설은 마지막 장에서 인물의 긴 여정을 돌아보게 합니다. 희곡은 막이 내려도 관객의 질문 속에서 계속되고, 연극은 커튼콜에서 모든 역할의 노력을 함께 기억합니다. 음악도 마지막 음이 멈춘 뒤 남는 울림이 작품의 진짜 힘이 됩니다.

여러분의 34차시 학습도 오늘로 한 막을 내리지만, 배움은 끝나지 않습니다. 문제를 정의하고, 설계하고, 구현하고, 실패를 고치고, 다른 사람의 의견을 받아들이며 여러분은 처음보다 훨씬 단단한 문제 해결자가 되었습니다.

그리고 발표 파일 이름을 ‘최종_진짜최종_마지막_수정없음_수정본’으로 저장했다면 버전 관리 성찰 한 줄은 이미 완성된 셈입니다. 오늘의 발표와 성찰이 하나의 종착점이 아니라 더 큰 질문과 꿈으로 출발하는 멋진 첫 장이 되기를 바랍니다.
