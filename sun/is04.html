학습할 개념

  • 중복 부분 문제: 같은 입력 상태가 반복 등장하는 구조
  • 메모이제이션: 계산 결과를 저장하고 재사용하는 최적화
  • 상태: 하나의 부분 문제를 구별하는 입력값
  • 캐시 적중: 저장된 결과를 다시 계산하지 않고 사용하는 경우
  • 시간-공간 절충: 실행 시간을 줄이기 위해 저장 공간을 사용하는 관계
개념 일반화
중복 부분 문제가 있는 알고리즘은 상태별 결과를 저장하고 재사용함으로써 계산 횟수와 실행 시간을 줄일 수 있지만, 그 대가로 추가 저장 공간을 사용한다.

Ⅲ. 수업 시작 멘트

“소크라테스는 이미 답한 질문을 다시 묻기보다 더 깊은 질문으로 나아가게 했고, 베이컨은 지식이 힘이라고 말했습니다. 뉴턴은 이전 과학자들의 성과 위에 서서 더 멀리 보았으며, 가우스는 같은 계산을 반복하기보다 구조를 발견해 한 번에 해결했습니다. 오늘의 메모이제이션도 같습니다. 한 번 힘들게 계산한 답을 잊어버리지 않고 다음 문제에 활용하는 지혜입니다. 매번 비밀번호를 새로 외우듯 같은 피보나치를 다시 계산하면 컴퓨터도 ‘아까 했잖아요’라고 말하고 싶을 겁니다. 오늘은 컴퓨터에게 기억력을 선물하고, 계산의 낭비를 지식의 재사용으로 바꾸어 봅시다.”

예제 5

리스트를 이용한 메모이제이션

리스트인덱스 저장
def fibonacci(n, memo=None):
    if memo is None:
        memo = [-1] * (n + 1)
        memo[0] = 0
        if n >= 1:
            memo[1] = 1

    if memo[n] != -1:
        return memo[n]

    memo[n] = fibonacci(n - 1, memo) + fibonacci(n - 2, memo)
    return memo[n]

print(fibonacci(40))
특징
입력값이 0부터 연속된 정수라면 리스트 인덱스를 활용해 빠르게 접근할 수 있습니다.
주의
기본 매개변수에 빈 리스트를 직접 작성하면 호출 사이에 값이 공유될 수 있으므로 memo=None 방식이 안전합니다.

예제 8

격자 경로 수 구하기

2차원 상태튜플 키
memo = {}

def count_paths(row, col):
    if row == 0 or col == 0:
        return 1

    key = (row, col)

    if key in memo:
        return memo[key]

    memo[key] = (
        count_paths(row - 1, col)
        + count_paths(row, col - 1)
    )
    return memo[key]

print(count_paths(10, 10))

메모의 키

상태가 두 값으로 결정되므로 (row, col) 튜플을 딕셔너리 키로 사용합니다.

같은 좌표의 경로 수를 다시 계산하지 않습니다.

예제 9

최소 비용 경로 재귀함수

최솟값부분 문제
cost = [1, 3, 2, 5, 4, 2]
memo = {}

def min_cost(index):
    if index == 0:
        return cost[0]
    if index == 1:
        return cost[1]

    if index in memo:
        return memo[index]

    memo[index] = cost[index] + min(
        min_cost(index - 1),
        min_cost(index - 2)
    )
    return memo[index]

print("최소 비용:", min_cost(len(cost) - 1))
분석
현재 위치까지 오는 최소 비용은 직전 칸과 두 칸 전 위치 중 더 작은 비용을 선택해 계산합니다.

예제 10

일반 재귀와 메모이제이션 성능 종합 비교

실행 시간호출 횟수종합 실습
import time

normal_calls = 0
memo_calls = 0

def fibonacci_normal(n):
    global normal_calls
    normal_calls += 1

    if n < 2:
        return n

    return (
        fibonacci_normal(n - 1)
        + fibonacci_normal(n - 2)
    )

memo = {0: 0, 1: 1}

def fibonacci_memo(n):
    global memo_calls
    memo_calls += 1

    if n in memo:
        return memo[n]

    memo[n] = (
        fibonacci_memo(n - 1)
        + fibonacci_memo(n - 2)
    )
    return memo[n]

n = 30

start = time.perf_counter()
normal_result = fibonacci_normal(n)
normal_time = time.perf_counter() - start

start = time.perf_counter()
memo_result = fibonacci_memo(n)
memo_time = time.perf_counter() - start

print("[일반 재귀]")
print("결과:", normal_result)
print("호출 횟수:", normal_calls)
print(f"실행 시간: {normal_time:.6f}초")

print("\n[메모이제이션]")
print("결과:", memo_result)
print("호출 횟수:", memo_calls)
print(f"실행 시간: {memo_time:.6f}초")
