예제 1

1부터 n까지의 합

n-1 문제선형 점화식
def recursive_sum(n):
    if n == 1:
        return 1

    return n + recursive_sum(n - 1)

print(recursive_sum(5))

점화식

S(1) = 1
S(n) = n + S(n-1)

도식

S(5)
└─ 5 + S(4)
       └─ 4 + S(3)
              └─ 3 + S(2)
                     └─ 2 + S(1)
크기 변화
입력 크기가 매 호출마다 1씩 감소하므로 5→4→3→2→1의 관계를 가집니다.

예제 5

이진 탐색의 입력 절반 분해

n/2 문제로그 단계
def binary_search(data, target, left, right):
    if left > right:
        return -1

    middle = (left + right) // 2

    if data[middle] == target:
        return middle

    if data[middle] < target:
        return binary_search(
            data, target, middle + 1, right
        )

    return binary_search(
        data, target, left, middle - 1
    )

numbers = list(range(1, 33))
print(binary_search(
    numbers, 29, 0, len(numbers) - 1
))

시간 점화식

T(1) = c
T(n) = T(n/2) + c

크기 변화

32 → 16 → 8 → 4 → 2 → 1

입력이 두 배가 되어도 탐색 단계는 약 한 단계만 증가합니다.

예제 6

빠른 거듭제곱의 절반 분해

분할 정복짝수·홀수
def fast_power(base, exponent):
    if exponent == 0:
        return 1

    half = fast_power(
        base,
        exponent // 2
    )

    if exponent % 2 == 0:
        return half * half

    return base * half * half

print(fast_power(2, 13))
관계
지수 n을 매번 절반으로 줄입니다. 짝수이면 절반 결과를 제곱하고, 홀수이면 밑을 한 번 더 곱합니다.
P(a, 0) = 1
P(a, n) = P(a, n/2)²              (n이 짝수)
P(a, n) = a × P(a, floor(n/2))²   (n이 홀수)

예제 7

병합 정렬의 두 부분 문제와 결합

두 절반병합 비용
def merge_sort(data):
    if len(data) <= 1:
        return data

    middle = len(data) // 2

    left = merge_sort(data[:middle])
    right = merge_sort(data[middle:])

    result = []
    i = j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    result.extend(left[i:])
    result.extend(right[j:])

    return result

print(merge_sort([7, 3, 8, 2, 5, 1]))

점화식

T(1) = c
T(n) = 2T(n/2) + n

절반 크기 문제 두 개와 n개 원소 병합 작업으로 구성됩니다.

분해 도식

│  ├─ [7]
│  └─ [3]
└─ [8,2]
   ├─ [8]
   └─ [2]

예제 8

배열 최대값을 절반으로 나누어 구하기

최댓값두 결과 결합
def divide_max(data, left, right):
    if left == right:
        return data[left]

    middle = (left + right) // 2

    left_max = divide_max(
        data, left, middle
    )
    right_max = divide_max(
        data, middle + 1, right
    )

    return max(left_max, right_max)

numbers = [12, 5, 19, 3, 21, 8]
print(divide_max(
    numbers, 0, len(numbers) - 1
))
결합
왼쪽 절반의 최대값과 오른쪽 절반의 최대값을 비교하면 전체 최대값을 얻을 수 있습니다.
M(1) = 원소값
M(n) = max(M(왼쪽 절반), M(오른쪽 절반))

T(n) = 2T(n/2) + c

예제 10

점화식과 입력 크기 관계 종합 분석

종합 비교도식 출력
def show_size_change(
    n,
    rule,
    minimum=1
):
    sizes = [n]

    while n > minimum:
        if rule == "minus_one":
            n -= 1

        elif rule == "half":
            n //= 2

        else:
            raise ValueError(
                "지원하지 않는 규칙입니다."
            )

        sizes.append(max(n, minimum))

    return sizes

print(
    "n-1 감소:",
    show_size_change(8, "minus_one")
)

print(
    "절반 감소:",
    show_size_change(32, "half")
)
문제부분 문제 관계대표 점화식입력 크기 도식
자연수 합·팩토리얼크기 n-1 한 개T(n)=T(n-1)+cn→n-1→…→1
피보나치·계단n-1, n-2 두 개T(n)=T(n-1)+T(n-2)+c두 갈래 호출 트리
이진 탐색·빠른 거듭제곱크기 n/2 한 개T(n)=T(n/2)+cn→n/2→n/4→…
병합 정렬크기 n/2 두 개T(n)=2T(n/2)+n이진 분해 후 병합
하노이 탑크기 n-1 두 번H(n)=2H(n-1)+1이동 문제가 두 번 반복
설명 예시
“병합 정렬은 입력 크기 n인 배열을 n/2 크기의 두 부분 배열로 나눈다. 두 부분 배열을 각각 정렬한 뒤 n개 원소를 비교하며 병합한다. 따라서 실행 시간은 두 개의 절반 크기 문제와 선형 병합 비용의 합인 T(n)=2T(n/2)+n으로 표현할 수 있다.”
