예제 4

여러 값을 반환하는 함수

튜플 반환언패킹데이터 영역
def analyze_scores(scores):
    total = sum(scores)
    average = total / len(scores)
    highest = max(scores)
    lowest = min(scores)
    return total, average, highest, lowest

scores = [85, 92, 78, 96, 89]
total, avg, high, low = analyze_scores(scores)

print("합계:", total)
print("평균:", avg)
print("최고:", high)
print("최저:", low)

문제 해결 연결

  1. 문제 이해: 점수의 특징을 요약한다.
  2. 데이터 분석: 합계·평균·최댓값·최솟값을 선택한다.
  3. 알고리즘: 네 값을 순서대로 계산한다.
  4. 함수 구현: 계산 결과를 한 번에 반환한다.
핵심 파이썬 함수는 쉼표로 구분하여 여러 값을 반환할 수 있으며, 호출할 때 여러 변수로 나누어 저장할 수 있습니다.

def mask_name(name):
    if len(name) <= 1:
        return "*"
    return name[0] + "*" * (len(name) - 1)

students = ["김민준", "이서연", "박준"]

def recommend_activity(temperature, rain):

오류 코드

def count_even(numbers):
    count = 0
    for number in numbers:
        if number % 2 == 0:
            count += 1
        return count

print(count_even([2, 4, 5, 8]))

수정 코드

def count_even(numbers):
    count = 0
    for number in numbers:
        if number % 2 == 0:

예제 10

종합 문제: 교실 환경 진단 프로그램

문제 분해함수 조합데이터알고리즘
def check_temperature(value):
    if 20 <= value <= 26:
        return "적정"
    return "조절 필요"

def check_co2(value):
    if value < 1000:
        return "양호"
    return "환기 필요"

def make_report(temperature, co2):
    temp_result = check_temperature(temperature)
    co2_result = check_co2(co2)

    return {
        "온도": temp_result,
        "이산화탄소": co2_result
    }

report = make_report(28, 1350)

for item, result in report.items():
    print(item, ":", result)

실행 결과

온도 : 조절 필요
이산화탄소 : 환기 필요

문제 해결 절차
