6. 학습할 코드 1개

def get_grade(score):
    if not isinstance(score, (int, float)):
        return "입력 오류"

    if score < 0 or score > 100:
        return "범위 오류"

    if score >= 90:
        return "A"
    elif score >= 80:
        return "B"
    elif score >= 70:
        return "C"
    elif score >= 60:
        return "D"
    return "F"


test_cases = [
    {"input": 100, "expected": "A"},
    {"input": 90, "expected": "A"},
    {"input": 89, "expected": "B"},
    {"input": 60, "expected": "D"},
    {"input": 59, "expected": "F"},
    {"input": -1, "expected": "범위 오류"},
    {"input": 101, "expected": "범위 오류"},
    {"input": "90", "expected": "입력 오류"}
]

passed = 0

for case in test_cases:
    actual = get_grade(case["input"])
    success = actual == case["expected"]

    if success:
        passed += 1

    print(
        case["input"],
        case["expected"],
        actual,
        "통과" if success else "실패"
    )

print("통과:", passed, "/", len(test_cases))
코드 요소상세 설명
isinstance입력값이 정수 또는 실수인지 검사해 잘못된 자료형을 구분한다.
score < 0 or score > 100허용 범위를 벗어난 값을 범위 오류로 처리한다.
>=90, 80, 70, 60과 같은 경계값을 해당 등급에 포함한다.
test_cases입력과 예상 결과를 딕셔너리로 묶은 자동 테스트 목록이다.
actual프로그램을 실제 실행해 얻은 결과이다.
success예상 결과와 실제 결과가 같은지를 나타내는 논리값이다.
passed통과한 테스트 수를 누적한다.

디버깅 실습: score >= 90을 score > 90으로 바꿔 90점 테스트가 실패하는지 확인한 뒤 다시 수정한다.
