6. 학습할 코드 1개

스택으로 괄호를 검사하고 큐로 작업을 처리하는 통합 프로그램

from collections import deque


def is_balanced(expression):
    """스택을 이용해 괄호의 짝과 순서를 검사한다."""
    stack = []
    pairs = {")": "(", "]": "[", "}": "{"}

    for char in expression:
        if char in "([{":
            stack.append(char)
        elif char in ")]}":
            if not stack:
                return False
            if stack.pop() != pairs[char]:
                return False

    return len(stack) == 0


def process_jobs(jobs):
    """큐를 이용해 입력된 순서대로 작업을 처리한다."""
    queue = deque(jobs)
    completed = []

    while queue:
        current_job = queue.popleft()
        completed.append(current_job)
        print("처리 완료:", current_job)

    return completed


expression = input("검사할 괄호식을 입력하세요: ")
jobs = ["보고서 출력", "그래프 출력", "평가표 출력"]

if is_balanced(expression):
    print("괄호가 올바릅니다.")
else:
    print("괄호의 짝이나 순서가 올바르지 않습니다.")

result = process_jobs(jobs)
print("전체 처리 순서:", result)

상세 설명

코드 요소설명
stack = []여는 괄호를 저장한다. 가장 최근에 들어온 여는 괄호를 먼저 비교해야 하므로 스택이 적합하다.
pairs닫는 괄호를 키로, 대응하는 여는 괄호를 값으로 저장해 짝을 빠르게 확인한다.
stack.append(char)여는 괄호를 스택의 맨 위에 추가한다.
stack.pop()가장 최근 여는 괄호를 꺼내 현재 닫는 괄호와 짝이 맞는지 검사한다.
return len(stack) == 0모든 문자를 확인한 뒤 스택이 비어 있어야 닫히지 않은 괄호가 없다는 뜻이다.
deque(jobs)작업 목록을 큐로 만든다. 리스트의 앞 원소 삭제보다 효율적으로 처리할 수 있다.
popleft()큐의 맨 앞 작업을 제거하고 반환하여 선입선출을 구현한다.
while queue큐에 작업이 남아 있는 동안 모든 작업을 순서대로 처리한다.

실행 예: 입력이 {[()]}이면 괄호가 올바르다고 출력하며, 작업은 ‘보고서 출력 → 그래프 출력 → 평가표 출력’ 순서로 처리된다.
