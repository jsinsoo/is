입력·처리·출력 함수 분리
위치·기본 매개변수 활용 예제 10선

프로그램을 입력 함수, 처리 함수, 출력 함수로 나누고 위치 인수와 기본 매개변수를 활용하여 재사용성과 수정 편의성을 높이는 파이썬 실습 자료입니다.

핵심 개념1. 기본 구조2. 사칙연산 3. 성적 처리4. 기본 매개변수5. 위치 인수 6. 단위 변환7. 주문 계산8. 문자열 처리 9. 데이터 요약10. 종합 프로그램

1. 입력·처리·출력 함수 분리

입력 함수사용자, 파일, 센서 등에서 필요한 값을 받아 정리한 뒤 반환합니다.
처리 함수입력받은 자료를 계산·분류·변환하여 결과를 반환합니다.
출력 함수처리 결과를 화면이나 파일에 알맞은 형식으로 표현합니다.
위치 인수는 함수 정의에 적힌 매개변수의 순서에 맞춰 값을 전달하는 방식입니다. 기본 매개변수는 호출할 때 값을 생략하면 미리 정해 둔 값을 사용하는 매개변수입니다. 기본값이 있는 매개변수는 일반 매개변수보다 뒤에 작성해야 합니다.
예제 1

입력·처리·출력의 기본 구조

함수 분리반환값재사용
def input_data():
    name = input("이름: ")
    age = int(input("나이: "))
    return name, age

def process_data(name, age):
    next_age = age + 1
    return name, next_age

def output_data(name, next_age):
    print(f"{name}님의 내년 나이는 {next_age}세입니다.")

name, age = input_data()
name, next_age = process_data(name, age)
output_data(name, next_age)

구조 분석

  1. input_data(): 이름과 나이를 입력받습니다.
  2. process_data(): 내년 나이를 계산합니다.
  3. output_data(): 결과 문장을 출력합니다.
재사용 점검
처리 함수는 직접 입력하지 않고 전달받은 값만 계산하므로 다른 프로그램에서도 사용할 수 있습니다.
예제 2

사칙연산 프로그램 함수 분리

위치 인수조건문기본 매개변수
def input_numbers():
    a, b = map(float, input("두 수: ").split())
    return a, b

def calculate(a, b, operator="+"):
    if operator == "+":
        return a + b
    elif operator == "-":
        return a - b
    elif operator == "*":
        return a * b
    elif operator == "/":
        return a / b if b != 0 else None

def output_result(result, title="계산 결과"):
    if result is None:
        print("계산할 수 없습니다.")
    else:
        print(f"{title}: {result}")

x, y = input_numbers()
result = calculate(x, y)
output_result(result)

기본값 사용

calculate(x, y)처럼 연산자를 생략하면 기본값 "+"가 사용됩니다.

output_result(result)에서 제목을 생략하면 "계산 결과"가 출력됩니다.

두 수: 7 3
계산 결과: 10.0
예제 3

학생 성적 처리 프로그램

리스트평균등급 분류
def input_scores():
    scores = list(map(int, input("점수들을 입력하세요: ").split()))
    return scores

def calculate_average(scores):
    return sum(scores) / len(scores)

def get_grade(average):
    if average >= 90:
        return "A"
    elif average >= 80:
        return "B"
    elif average >= 70:
        return "C"
    else:
        return "D"

def output_report(average, grade, digits=1):
    print(f"평균: {average:.{digits}f}")
    print(f"등급: {grade}")

scores = input_scores()
average = calculate_average(scores)
grade = get_grade(average)
output_report(average, grade)
기본 매개변수
digits=1은 평균을 소수점 한 자리까지 표시합니다. output_report(average, grade, 2)로 호출하면 두 자리까지 표시합니다.

def input_name():

def make_greeting(name, greeting="안녕하세요", punctuation="!"):

예제 6

길이 단위 변환 프로그램

단위 변환기본 단위조건 분기
def input_length():
    return float(input("길이: "))

def convert_length(value, unit="cm"):
    if unit == "cm":
        return value / 100
    elif unit == "mm":
        return value / 1000
    elif unit == "km":
        return value * 1000
    else:
        return value

def output_length(meters, digits=3):
    print(f"{meters:.{digits}f} m")

length = input_length()
meters = convert_length(length)
output_length(meters)
재사용 예
convert_length(2500, "mm"), convert_length(1.5, "km")처럼 다양한 입력 단위를 같은 함수로 변환할 수 있습니다.
예제 7

상품 주문 금액 계산

가격 계산할인율배송비
def input_order():
    price = int(input("상품 가격: "))
    quantity = int(input("수량: "))
    return price, quantity

def calculate_total(price, quantity, discount=0, delivery=3000):
    subtotal = price * quantity
    discount_amount = subtotal * discount
    return subtotal - discount_amount + delivery

def output_total(total, currency="원"):
    print(f"최종 결제 금액: {total:,.0f}{currency}")

price, quantity = input_order()
total = calculate_total(price, quantity)
output_total(total)

활용 예

calculate_total(10000, 3)
기본 할인율 0, 기본 배송비 3000원

calculate_total(10000, 3, 0.1)
10% 할인, 기본 배송비 적용

calculate_total(10000, 3, 0.1, 0)
10% 할인, 무료 배송

예제 8

문자열 정리 프로그램

문자열 처리대소문자기본 구분자
def input_text():
    return input("문장: ")

def process_text(text, separator=" ", upper=False):
    words = text.strip().split(separator)
    result = "-".join(words)

    if upper:
        result = result.upper()

    return result

def output_text(result, label="변환 결과"):
    print(f"{label}: {result}")

text = input_text()
result = process_text(text)
output_text(result)
위치 인수 활용
process_text("apple,banana,cherry", ",", True)는 쉼표로 나누고 대문자로 변환합니다.
예제 9

데이터 요약 함수 조합

데이터딕셔너리출력 형식
def input_data():
    return list(map(float, input("자료: ").split()))

def summarize(data, include_range=True):
    result = {
        "개수": len(data),
        "평균": sum(data) / len(data),
        "최댓값": max(data),
        "최솟값": min(data)
    }

    if include_range:
        result["범위"] = max(data) - min(data)

    return result

def output_summary(summary, digits=2):
    for key, value in summary.items():
        if isinstance(value, float):
            print(f"{key}: {value:.{digits}f}")
        else:
            print(f"{key}: {value}")

data = input_data()
summary = summarize(data)
output_summary(summary)

기본 매개변수의 의미

include_range=True이므로 기본적으로 범위를 계산합니다.

summarize(data, False)로 호출하면 범위를 제외할 수 있습니다.

digits=2는 실수 결과를 소수점 둘째 자리까지 표시합니다.

예제 10

종합: 교실 환경 상태 보고서

종합 실습함수 조합기본 기준값재사용
def input_environment():
    temperature = float(input("온도(℃): "))
    humidity = float(input("습도(%): "))
    co2 = int(input("이산화탄소(ppm): "))
    return temperature, humidity, co2

def check_temperature(value, minimum=20, maximum=26):
    if minimum <= value <= maximum:
        return "적정"
    return "조절 필요"

def check_humidity(value, minimum=40, maximum=60):
    if minimum <= value <= maximum:
        return "적정"
    return "조절 필요"

def check_co2(value, limit=1000):
    if value < limit:
        return "양호"
    return "환기 필요"

def make_report(temperature, humidity, co2):
    return {
        "온도": check_temperature(temperature),
        "습도": check_humidity(humidity),
        "이산화탄소": check_co2(co2)
    }

def output_report(report, title="교실 환경 진단"):
    print(f"\n[{title}]")
    for item, result in report.items():
        print(f"{item}: {result}")

temperature, humidity, co2 = input_environment()
report = make_report(temperature, humidity, co2)
output_report(report)

입력 함수

세 가지 환경값을 입력받아 튜플 형태로 반환합니다.

처리 함수

온도·습도·이산화탄소를 각각 판단하고 보고서 딕셔너리로 묶습니다.

출력 함수

보고서 제목과 각 항목을 일정한 형식으로 출력합니다.

재사용 방법

check_co2(value, 800)처럼 기준값을 바꾸어 다른 환경에도 적용할 수 있습니다.

마무리 진단 문제

1. 입력 함수가 계산까지 모두 담당하면 어떤 문제가 생길 수 있는가?
정답 보기입력 방식이 바뀔 때 계산 코드까지 수정해야 하므로 재사용성과 유지보수성이 낮아집니다.
2. def f(a, b=10):에서 f(3)을 호출하면 b의 값은 무엇인가?
정답 보기기본값인 10입니다.
3. make_greeting("민수", "반갑습니다")에서 두 문자열은 어떤 방식으로 전달되는가?
정답 보기매개변수의 순서에 따라 전달되는 위치 인수입니다.
4. 기본값이 없는 매개변수와 기본값이 있는 매개변수 중 어느 것을 먼저 작성해야 하는가?
정답 보기기본값이 없는 일반 매개변수를 먼저 작성해야 합니다.
5. 처리 함수가 결과를 직접 출력하지 않고 반환하도록 작성하면 어떤 장점이 있는가?
정답 보기반환값을 저장·재계산·파일 저장·다른 함수 전달 등 여러 방식으로 재사용할 수 있습니다.
