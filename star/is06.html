예제 1

하노이 탑 기본 재귀함수

분할 정복원판 이동
def hanoi(n, start, target, helper):
    if n == 1:
        print(f"{start} → {target}")
        return

    hanoi(n - 1, start, helper, target)
    print(f"{start} → {target}")
    hanoi(n - 1, helper, target, start)

hanoi(3, "A", "C", "B")

문제 분해

  1. 위쪽 n-1개 원판을 보조 기둥으로 옮깁니다.
  2. 가장 큰 원판을 목표 기둥으로 옮깁니다.
  3. 보조 기둥의 n-1개를 목표 기둥으로 옮깁니다.
종료 조건
원판이 1개이면 더 나눌 필요 없이 시작 기둥에서 목표 기둥으로 바로 옮깁니다.
예제 2

하노이 탑 재귀 호출 추적

호출 깊이과정 추적
def hanoi_trace(n, start, target, helper, depth=0):
    indent = "  " * depth
    print(f"{indent}hanoi({n}, {start}, {target}, {helper})")

    if n == 1:
        print(f"{indent}이동: {start} → {target}")
        return

    hanoi_trace(n - 1, start, helper, target, depth + 1)
    print(f"{indent}이동: {start} → {target}")
    hanoi_trace(n - 1, helper, target, start, depth + 1)

hanoi_trace(3, "A", "C", "B")
관찰
들여쓰기가 깊어질수록 더 작은 원판 문제를 해결하고 있음을 확인할 수 있습니다.
예제 3

하노이 탑 최소 이동 횟수

점화식이동 횟수
def hanoi_count(n):
    if n == 1:
        return 1

    return 2 * hanoi_count(n - 1) + 1

for n in range(1, 6):
    print(n, "개:", hanoi_count(n), "회")

계산 관계

T(1) = 1
T(2) = 2×T(1)+1 = 3
T(3) = 2×T(2)+1 = 7
T(4) = 15
T(5) = 31
핵심
n개 원판을 옮기려면 n-1개를 두 번 옮기고 가장 큰 원판을 한 번 옮깁니다.
예제 4

깊이 우선 방식의 미로 탐색

DFS방문 표시미로
maze = [
    [0, 1, 0, 0],
    [0, 0, 0, 1],
    [1, 0, 1, 0],
    [0, 0, 0, 0]
]

rows = len(maze)
cols = len(maze[0])
visited = [[False] * cols for _ in range(rows)]

def solve(row, col):
    if row < 0 or row >= rows:
        return False
    if col < 0 or col >= cols:
        return False
    if maze[row][col] == 1 or visited[row][col]:
        return False

    if row == rows - 1 and col == cols - 1:
        return True

    visited[row][col] = True

    return (
        solve(row + 1, col)
        or solve(row - 1, col)
        or solve(row, col + 1)
        or solve(row, col - 1)
    )

print("탈출 가능:", solve(0, 0))
종료 조건
배열 밖, 벽, 방문한 칸이면 실패하고, 출구에 도착하면 성공을 반환합니다.
예제 5

미로 경로 저장과 복원

백트래킹경로 복원
maze = [
    [0, 1, 0, 0],
    [0, 0, 0, 1],
    [1, 0, 1, 0],
    [0, 0, 0, 0]
]

rows = len(maze)
cols = len(maze[0])
visited = set()
path = []

def find_path(row, col):
    if not (0 <= row < rows and 0 <= col < cols):
        return False
    if maze[row][col] == 1 or (row, col) in visited:
        return False

    visited.add((row, col))
    path.append((row, col))

    if (row, col) == (rows - 1, cols - 1):
        return True

    directions = [(1,0), (-1,0), (0,1), (0,-1)]

    for dr, dc in directions:
        if find_path(row + dr, col + dc):
            return True

    path.pop()
    return False

if find_path(0, 0):
    print("경로:", path)
else:
    print("경로 없음")
백트래킹
현재 선택이 출구로 이어지지 않으면 path.pop()으로 마지막 위치를 제거하고 이전 상태로 돌아갑니다.
예제 6

미로의 모든 경로 찾기

모든 해방문 해제
maze = [
    [0, 0, 0],
    [0, 1, 0],
    [0, 0, 0]
]

rows = len(maze)
cols = len(maze[0])
visited = set()
all_paths = []

def find_all_paths(row, col, path):
    if not (0 <= row < rows and 0 <= col < cols):
        return
    if maze[row][col] == 1 or (row, col) in visited:
        return

    visited.add((row, col))
    path.append((row, col))

    if (row, col) == (rows - 1, cols - 1):
        all_paths.append(path.copy())
    else:
        for dr, dc in [(1,0), (-1,0), (0,1), (0,-1)]:
            find_all_paths(row + dr, col + dc, path)

    path.pop()
    visited.remove((row, col))

find_all_paths(0, 0, [])

print("경로 수:", len(all_paths))
for path in all_paths:
    print(path)
복원 이유
한 경로 탐색이 끝난 뒤 방문 표시를 해제해야 다른 경로에서도 같은 칸을 사용할 수 있습니다.
예제 7

재귀함수로 순열 생성

순열선택·취소
def permutations(items, selected):
    if len(selected) == len(items):
        print(selected)
        return

    for item in items:
        if item not in selected:
            selected.append(item)
            permutations(items, selected)
            selected.pop()

permutations(["A", "B", "C"], [])

예제 8

n개 중 r개 조합 생성

조합시작 인덱스
def combinations(items, r, start=0, selected=None):
    if selected is None:
        selected = []

    if len(selected) == r:
        print(selected)
        return

    for index in range(start, len(items)):
        selected.append(items[index])
        combinations(items, r, index + 1, selected)
        selected.pop()

combinations(["A", "B", "C", "D"], 2)
순열과 차이
조합은 순서를 구분하지 않으므로 이전에 선택한 위치보다 뒤쪽 원소만 탐색합니다.
예제 9

포함·제외 방식의 부분집합 생성

부분집합이진 선택
def subsets(items, index=0, selected=None):
    if selected is None:
        selected = []

    if index == len(items):
        print(selected)
        return

    # 현재 원소를 포함
    selected.append(items[index])
    subsets(items, index + 1, selected)

    # 현재 원소를 제외
    selected.pop()
    subsets(items, index + 1, selected)

subsets([1, 2, 3])

문제 분해

각 원소마다 두 가지 선택을 합니다.

  1. 현재 원소를 부분집합에 포함한다.
  2. 현재 원소를 부분집합에서 제외한다.

원소가 n개이면 부분집합은 2ⁿ개입니다.

마무리 진단 문제

1. 하노이 탑에서 원판 n개 문제는 어떤 작은 문제로 분해되는가?
정답 보기n-1개 원판을 옮기는 문제 두 번과 가장 큰 원판을 옮기는 문제 한 번으로 분해됩니다.
2. 미로 탐색에서 방문 표시가 필요한 이유는 무엇인가?
정답 보기같은 칸을 반복 방문하여 무한 재귀에 빠지는 것을 막기 위해서입니다.
3. 백트래킹에서 pop()의 역할은 무엇인가?
정답 보기현재 선택을 취소하여 이전 상태로 복원하고 다른 선택을 탐색하도록 합니다.
4. 조합 생성에서 시작 인덱스를 증가시키는 이유는 무엇인가?
정답 보기이미 선택한 원소보다 뒤쪽만 탐색하여 같은 조합의 순서만 다른 중복을 막기 위해서입니다.
5. 부분집합 생성에서 각 원소마다 고려하는 두 선택은 무엇인가?
정답 보기현재 원소를 포함하는 선택과 제외하는 선택입니다.
