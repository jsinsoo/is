1. 스택과 LIFO 원리

LIFO나중에 들어온 자료가 먼저 나오는 후입선출 방식입니다.
push스택의 맨 위에 새로운 자료를 추가합니다.
pop스택의 맨 위 자료를 제거하고 반환합니다.
peek맨 위 자료를 삭제하지 않고 확인합니다.
접시 쌓기, 브라우저 뒤로 가기, 문서 실행 취소, 함수 호출 기록은 스택의 대표 사례입니다. 파이썬 리스트에서는 맨 뒤를 스택의 꼭대기로 사용하면 append()와 pop()으로 효율적으로 구현할 수 있습니다.
연산리스트 구현자료 변화
push(x)stack.append(x)맨 뒤에 x 추가
pop()stack.pop()맨 뒤 자료 삭제·반환
peek()stack[-1]맨 뒤 자료 확인
is_empty()len(stack) == 0비어 있는지 확인

접시 쌓기로 LIFO 확인

plates = []

plates.append("파란 접시")
plates.append("노란 접시")
plates.append("빨간 접시")

예제 3

pop 함수 직접 구현

삭제 연산반환값
stack = [10, 20, 30]

def pop():
    if len(stack) == 0:
        return None

    value = stack[-1]
    del stack[-1]
    return value

print("삭제값:", pop())
print("삭제값:", pop())
print("현재 스택:", stack)
직접 구현
맨 위 값을 먼저 저장하고 del stack[-1]로 삭제한 뒤 저장값을 반환합니다. 실제로는 리스트의 pop()을 바로 사용할 수 있습니다.

stack = ["A", "B", "C"]

def peek():
    if len(stack) == 0:

    return stack[-1]

예제 5

Stack 클래스로 연산 통합

클래스캡슐화
class Stack:
    def __init__(self):
        self.items = []

    def push(self, value):
        self.items.append(value)

    def pop(self):
        if self.is_empty():
            return None
        return self.items.pop()

    def peek(self):
        if self.is_empty():
            return None
        return self.items[-1]

    def is_empty(self):
        return len(self.items) == 0

    def size(self):
        return len(self.items)

stack = Stack()
stack.push(100)
stack.push(200)

print("peek:", stack.peek())
print("pop:", stack.pop())
print("크기:", stack.size())
장점
스택 자료와 연산을 하나의 클래스로 묶어 여러 스택 객체를 독립적으로 만들 수 있습니다.

class Stack:
    def __init__(self):
        self.items = []

    def pop(self):
        if not self.items:
            raise IndexError(
                "빈 스택에서는 pop할 수 없습니다."
            )
        return self.items.pop()

stack = Stack()

try:
    print(stack.pop())
except IndexError as error:
    print("오류:", error)

예제 7

문서 편집 실행 취소

실행 취소상태 저장
text = ""
history = []

def write(new_text):
    global text

    history.append(text)
    text += new_text
    print("현재 문서:", text)

def undo():
    global text

    if not history:
        print("취소할 작업이 없습니다.")
        return

    text = history.pop()
    print("실행 취소:", text)

write("안녕")
write("하세요")
write("!")
undo()
undo()
LIFO 적용
최근 상태를 스택의 맨 위에 저장하므로 가장 최근 작업부터 역순으로 취소할 수 있습니다.
예제 8

스택으로 문자열 뒤집기

문자열역순 출력
def reverse_text(text):
    stack = []

    for character in text:
        stack.append(character)

    result = ""

    while stack:
        result += stack.pop()

    return result

print(reverse_text("PYTHON"))

예제 9

괄호 짝 검사

괄호 검사응용
def check_parentheses(expression):
    stack = []

    for character in expression:
        if character == "(":
            stack.append(character)

        elif character == ")":
            if not stack:
                return False
            stack.pop()

    return len(stack) == 0

tests = [
    "(a+b)*(c-d)",
    "((a+b)",
    "a+b)",
    "(a+(b*c))"
]

for test in tests:
    print(test, "→", check_parentheses(test))

예제 10

명령어 기반 스택 프로그램

종합 실습push·pop·peek
stack = []

def push(value):
    stack.append(value)

def pop():
    if not stack:
        return "EMPTY"
    return stack.pop()

def peek():
    if not stack:
        return "EMPTY"
    return stack[-1]

while True:
    command = input("명령: ").split()

    if command[0] == "push":
        push(command[1])
        print("스택:", stack)

    elif command[0] == "pop":
        print("삭제:", pop())

    elif command[0] == "peek":
        print("맨 위:", peek())

    elif command[0] == "size":
        print("크기:", len(stack))

    elif command[0] == "quit":
        break

    else:
        print("알 수 없는 명령입니다.")
명령기능예시
push 값값을 맨 위에 추가push 10
pop맨 위 값을 삭제·출력pop
peek맨 위 값을 확인peek
size저장된 자료 수 출력size
quit프로그램 종료quit
