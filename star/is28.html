같은 소수 판별도 검사 범위에 따라 연산량이 달라집니다.

주제: 기본 판별과 제곱근 범위 판별의 연산 횟수 비교

def is_prime(n):
    if n < 2:
        return False

    divisor = 2
    while divisor * divisor <= n:
        if n % divisor == 0:
            return False
        divisor += 1

    return True

for number in range(2, 30):
    if is_prime(number):
        print(number, end=" ")
확장: 에라토스테네스의 체와 입력 크기별 실행 시간을 비교합니다.

관찰과 주제

함수값의 변화를 이용해 그래프의 증가·감소 구간을 자동으로 찾습니다.

주제: 이산적인 함수값 변화로 증가·감소 구간을 분석하는 프로그램

def f(x):
    return x ** 2 - 4 * x + 3

x_values = list(range(-2, 7))
y_values = [f(x) for x in x_values]

for i in range(len(x_values) - 1):
    change = y_values[i + 1] - y_values[i]
    state = "증가" if change > 0 else "감소" if change < 0 else "일정"
    print(x_values[i], "→", x_values[i + 1], state)

주제: 메뉴 유형별 평균 잔반량을 분석하여 잔반 감소 방안 제안

records = [
    ("한식", 18), ("면류", 9), ("튀김", 7),
    ("한식", 15), ("면류", 11), ("튀김", 8)
]

totals, counts = {}, {}

for menu, waste in records:
    totals[menu] = totals.get(menu, 0) + waste
    counts[menu] = counts.get(menu, 0) + 1

for menu in totals:
    print(menu, "평균 잔반:", totals[menu] / counts[menu])

예제 8

교내 장소 최소 이동 경로 안내

그래프BFS

관찰과 주제

신입생과 방문자가 특별실을 찾기 어려운 문제를 관찰합니다.

주제: 교내 장소를 그래프로 모델링한 최소 이동 경로 안내

from collections import deque

graph = {
    "교실": ["복도"],
    "복도": ["교실", "계단", "도서관"],
    "계단": ["복도", "과학실"],
    "도서관": ["복도", "과학실"],
    "과학실": ["계단", "도서관"]
}

def shortest_path(start, target):
    queue = deque([(start, [start])])
    visited = {start}

    while queue:
        place, path = queue.popleft()
        if place == target:
            return path

        for next_place in graph[place]:
            if next_place not in visited:
                visited.add(next_place)
                queue.append((next_place, path + [next_place]))

print(shortest_path("교실", "과학실"))

운동장 수업 전에 미세먼지 상태를 빠르게 판단할 필요가 있습니다.

pm25 = {
    "09시": 18, "10시": 26,
    "11시": 41, "12시": 37,
    "13시": 22
}

for time_value, concentration in pm25.items():
    if concentration <= 15:
        state = "매우 좋음"
    elif concentration <= 35:
        state = "활동 가능"
    else:
        state = "실내 활동 권장"

    print(time_value, concentration, state)

학습 시간과 집중도 변화 분석

관찰과 주제

공부 시간이 길수록 집중도가 항상 높아지는지 자기 기록으로 탐구합니다.

주제: 학습 시간·휴식 간격·집중도의 관계 분석

records = [
    (30, 8), (45, 9), (60, 7),
    (90, 5), (40, 8)
]

best = max(records, key=lambda item: item[1])
average_focus = sum(focus for _, focus in records) / len(records)

print("집중도가 가장 높은 시간:", best[0], "분")
print("평균 집중도:", average_focus)

1. 프로젝트 문제를 수치화해야 하는 이유는 무엇인가?
