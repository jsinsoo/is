6. 학습할 코드 1개

하나의 리스트 입력을 스택과 큐 방식으로 처리하기

class Stack:
    def __init__(self):
        self.items = []

    def push(self, value):
        self.items.append(value)

    def pop(self):
        if self.is_empty():
            return None
        return self.items.pop()

    def peek(self):
        if self.is_empty():
            return None
        return self.items[-1]

    def is_empty(self):
        return len(self.items) == 0


class Queue:
    def __init__(self):
        self.items = []

    def enqueue(self, value):
        self.items.append(value)

    def dequeue(self):
        if self.is_empty():
            return None
        return self.items.pop(0)

    def front(self):
        if self.is_empty():
            return None
        return self.items[0]

    def is_empty(self):
        return len(self.items) == 0


data = ["A", "B", "C"]

stack = Stack()
queue = Queue()

for value in data:
    stack.push(value)
    queue.enqueue(value)

print("스택에서 꺼내기:", stack.pop())
print("큐에서 꺼내기:", queue.dequeue())
print("스택의 다음 데이터:", stack.peek())
print("큐의 다음 데이터:", queue.front())

상세 설명

코드 요소설명
self.items = []스택과 큐의 데이터를 저장할 순차적 자료구조로 파이썬 리스트를 사용한다.
append(value)리스트의 끝에 데이터를 추가한다. 스택의 push와 큐의 enqueue에 공통으로 사용된다.
pop()리스트의 마지막 데이터를 제거하므로 스택의 후입선출 원리를 구현한다.
pop(0)리스트의 첫 번째 데이터를 제거하므로 큐의 선입선출 원리를 구현한다.
peek()스택의 맨 위 데이터를 제거하지 않고 확인한다.
front()큐의 맨 앞 데이터를 제거하지 않고 확인한다.
is_empty()비어 있는 구조에서 삭제를 시도할 때 오류가 발생하지 않도록 검사한다.

입력 순서가 A→B→C이면 스택에서 처음 꺼내는 값은 C이고, 큐에서 처음 꺼내는 값은 A이다. 같은 데이터라도 처리 규칙에 따라 결과 순서가 달라진다.
