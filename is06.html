6. 학습할 코드 1개

세 수의 합을 만족하는 모든 상태 탐색하기

target = int(input("목표 합을 입력하세요: "))
limit = int(input("각 수의 최댓값을 입력하세요: "))

solutions = []
checked_states = 0

for a in range(1, limit + 1):
    for b in range(1, limit + 1):
        for c in range(1, limit + 1):
            checked_states += 1

            if a + b + c == target:
                solutions.append((a, b, c))

print("탐색한 상태 수:", checked_states)
print("찾은 해의 개수:", len(solutions))

for solution in solutions:
    print(solution)

상세 설명

코드 요소설명
target세 수의 합이 도달해야 하는 목표값이다. 목표 상태 판정 기준이 된다.
limit각 변수 a, b, c가 가질 수 있는 탐색 범위의 상한이다.
solutions = []목표 조건을 만족한 상태를 모두 저장한다.
삼중 반복문a, b, c의 가능한 모든 조합을 생성하여 전체 상태 공간을 탐색한다.
checked_states += 1실제로 검사한 상태 수를 기록한다. 전체 탐색량을 확인할 수 있다.
a + b + c == target현재 상태가 목표 조건을 만족하는지 검사한다.
solutions.append((a,b,c))조건을 만족한 상태를 튜플로 저장한다.
limit³각 변수가 1부터 limit까지라면 전체 상태 수는 limit×limit×limit이다.

예: target=6, limit=4이면 (1,1,4), (1,2,3), (2,2,2) 등 조건을 만족하는 모든 순서쌍을 찾는다. 순서가 다른 경우를 서로 다른 상태로 본다는 점에 주의한다.
