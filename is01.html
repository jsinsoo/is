2. 학습할 코드

점수 목록과 기준 점수를 입력받아 평균, 통과 여부, 요약 문장을 반환하는 예제입니다. 함수 정의·호출·매개변수·반환값을 한 번에 학습하도록 설계했습니다.

student_score_analyzer.py
def analyze_scores(scores, pass_score=60):
    """점수 목록을 분석하여 평균, 통과 여부, 요약 문장을 반환한다."""
    if not scores:
        return 0, False, "점수 데이터가 없습니다."

    average = sum(scores) / len(scores)
    passed = average >= pass_score

    if average >= 90:
        level = "매우 우수"
    elif average >= 80:
        level = "우수"
    elif average >= 70:
        level = "보통"
    else:
        level = "보완 필요"

    message = f"평균 {average:.1f}점, 성취 수준: {level}"
    return average, passed, message


student_scores = [88, 92, 76, 95]
avg, is_passed, summary = analyze_scores(student_scores, 80)

print(f"평균: {avg:.1f}")
print(f"기준 통과: {is_passed}")
print(summary)
코드 요소상세 설명개념적 의미
def analyze_scores(...)함수의 이름과 입력 구조를 정의한다. scores는 점수 목록, pass_score는 기준 점수다.반복되는 분석 절차를 하나의 기능 단위로 추상화한다.
pass_score=60호출할 때 기준 점수를 생략하면 60을 사용하는 기본 매개변수다.기본값을 제공하면서도 필요할 때 동작을 변경할 수 있다.
if not scores빈 목록이 들어왔을 때 0으로 나누는 오류를 막는다.입력의 경계 조건을 고려하는 안전한 프로그램 설계다.
return average, passed, message평균, 통과 여부, 요약 문장을 호출한 위치로 돌려준다.함수의 결과를 다른 계산과 판단에 다시 사용할 수 있게 한다.
avg, is_passed, summary = ...함수가 반환한 세 값을 각각의 변수에 나누어 저장한다.함수 호출 결과가 프로그램의 다음 단계로 이어지는 데이터 흐름을 보여 준다.
예상 실행 결과
평균: 87.8
기준 통과: True
평균 87.8점, 성취 수준: 우수

시는 짧은 문장으로 마음을 움직이고, 소설은 인물에게 삶을 맡기며, 희곡과 연극은 대사 하나를 무대 위 행동으로 바꾸고, 음악은 반복되는 주제를 변주합니다. 함수도 같습니다. 짧은 이름 하나에 긴 행동을 담고, 필요할 때 다시 불러 새로운 장면을 만듭니다. 오늘 오류가 났다면 실패가 아니라 코드가 앙코르를 외친 것입니다. 내일의 여러분은 오늘 만든 함수를 호출해 한 단계 더 멀리 갈 것입니다.
