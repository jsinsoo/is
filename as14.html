1 연구 질문과 결과의 연결

결과가 질문에 어떻게 답하는가

개념 상세 설명: 분석 결과는 연구 질문에 대한 답을 뒷받침하는 근거로 사용해야 한다. 연구 질문, 사용한 지표, 핵심 수치와 해석을 한 흐름으로 연결하면 결과가 단순 나열에서 논리적 설명으로 발전한다.

research_question = "공부 시간이 늘수록 점수가 높아지는가?"
study_hours = [1, 2, 3, 4, 5]
scores = [58, 64, 71, 78, 86]

mean_gain = (scores[-1] - scores[0]) / (study_hours[-1] - study_hours[0])

print("연구 질문:", research_question)
print("시간 1시간 증가당 평균 점수 변화:", round(mean_gain, 2))
print("해석: 표본에서는 공부 시간이 늘수록 점수가 높아지는 경향이 나타났다.")
코드 상세 설명: 연구 질문을 먼저 제시하고, 시작점과 끝점의 변화량을 계산하여 결과를 질문과 연결한다. 마지막 문장은 ‘표본에서는’이라는 범위를 밝혀 과도한 일반화를 피한다.
[개념 1] 연구 질문과 결과 연결에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ⑤ — 질문, 근거와 해석이 한 흐름으로 연결되어야 한다.

2 기술과 해석의 구분

보이는 사실과 의미를 따로 본다

개념 상세 설명: 기술은 자료에서 직접 확인한 수치와 패턴이고, 해석은 그 패턴이 무엇을 의미하는지 설명하는 것이다. 기술과 해석을 구분하면 추측을 사실처럼 표현하는 오류를 줄일 수 있다.

temperatures = [24, 25, 27, 29, 30]
power_use = [120, 126, 139, 154, 166]

description = f"기온은 {temperatures[0]}℃에서 {temperatures[-1]}℃로 증가했고, 전력 사용량은 {power_use[0]}에서 {power_use[-1]}로 증가했다."
interpretation = "두 값은 함께 증가하는 경향을 보였으나, 이 결과만으로 기온이 전력 사용 증가의 유일한 원인이라고 단정할 수 없다."

print("기술:", description)
print("해석:", interpretation)
코드 상세 설명: 첫 문장은 관찰된 수치를 기술하고, 두 번째 문장은 그 의미와 해석의 한계를 설명한다.
[개념 2] 기술과 해석에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ④ — 보이는 사실과 그 의미에 대한 설명을 구분해야 한다.

3 상관관계와 인과관계

함께 변함과 원인 관계는 다르다

개념 상세 설명: 상관관계는 두 변수가 함께 변하는 정도를 뜻하고, 인과관계는 한 변수의 변화가 다른 변수의 변화를 일으킨다는 의미이다. 높은 상관계수만으로 인과를 증명할 수 없으며 시간 순서, 교란 요인 통제와 실험적 근거가 추가로 필요하다.

import pandas as pd

data = pd.DataFrame({
    "기온": [20, 22, 24, 26, 28, 30],
    "아이스크림판매": [45, 52, 63, 75, 88, 101]
})

correlation = data["기온"].corr(data["아이스크림판매"])

print("상관계수:", round(correlation, 3))
print("해석: 두 변수는 강한 양의 상관을 보인다.")
print("주의: 상관계수만으로 기온이 판매 증가의 직접 원인이라고 확정할 수는 없다.")
코드 상세 설명: 피어슨 상관계수를 계산하여 함께 변하는 정도를 확인한다. 마지막 문장은 상관과 인과를 구분한다.
[개념 3] 상관과 인과에 대한 옳은 설명 3개를 고르시오.
정답: ②, ③, ⑤ — 함께 변하는 현상과 원인 관계를 구분해야 한다.

4 교란 요인

두 변수 모두에 영향을 주는 숨은 요인

개념 상세 설명: 교란 요인은 원인으로 생각한 변수와 결과 변수 모두에 영향을 주어 관계를 왜곡할 수 있는 제3의 변수이다. 교란 요인을 찾아 측정하거나 조건을 나누어 분석하면 더 타당한 설명을 만들 수 있다.

records = [
    {"study": 2, "sleep": 5, "score": 60},
    {"study": 3, "sleep": 6, "score": 68},
    {"study": 4, "sleep": 8, "score": 80},
    {"study": 5, "sleep": 8, "score": 86}
]

high_sleep = [r["score"] for r in records if r["sleep"] >= 8]
low_sleep = [r["score"] for r in records if r["sleep"] < 8]

print("수면 8시간 이상 평균:", sum(high_sleep) / len(high_sleep))
print("수면 8시간 미만 평균:", sum(low_sleep) / len(low_sleep))
print("해석: 공부 시간뿐 아니라 수면 시간도 점수와 관련될 가능성이 있다.")
코드 상세 설명: 공부 시간과 점수 관계를 볼 때 수면 시간이 함께 영향을 줄 수 있음을 확인한다. 수면은 잠재적 교란 요인이다.
[개념 4] 교란 요인에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ④ — 숨은 제3의 변수가 관계를 왜곡할 수 있다.

5 대안 설명

한 결과를 설명하는 다른 가능성

개념 상세 설명: 대안 설명은 연구자가 처음 생각한 설명 외에 결과를 설명할 수 있는 다른 가능성이다. 계절, 측정 시점, 표본 구성, 선택 편향과 우연한 변동을 검토하면 결론을 더 신중하게 만들 수 있다.

observed = "환기 후 집중도 점수가 높아졌다."

alternatives = [
    "수업 내용의 난이도가 낮아졌을 수 있다.",
    "쉬는 시간 직후라 피로가 줄었을 수 있다.",
    "집중도 측정 방법이 달라졌을 수 있다.",
    "참여 학생 구성이 달라졌을 수 있다."
]

print("관찰 결과:", observed)
print("가능한 대안 설명")
for item in alternatives:
    print("-", item)
코드 상세 설명: 하나의 관찰 결과에 대해 여러 가능성을 목록으로 정리한다. 대안 설명을 검토하면 한 가지 원인을 성급하게 확정하지 않게 된다.
[개념 5] 대안 설명에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ⑤ — 다른 가능성을 검토해야 해석의 타당성이 높아진다.

6 효과 크기와 실질적 의미

차이가 있는가보다 얼마나 중요한가

개념 상세 설명: 결과 해석에서는 차이의 존재뿐 아니라 변화의 크기와 실제 의미를 함께 봐야 한다. 아주 작은 차이는 통계적으로 관찰되더라도 교육적·과학적·사회적으로 중요하지 않을 수 있다.

before = [70, 72, 74, 76, 78]
after = [72, 74, 76, 78, 80]

before_mean = sum(before) / len(before)
after_mean = sum(after) / len(after)
difference = after_mean - before_mean
rate = difference / before_mean * 100

print("개선 전 평균:", before_mean)
print("개선 후 평균:", after_mean)
print("평균 차이:", difference)
print("변화율:", round(rate, 2), "%")
코드 상세 설명: 평균 차이와 변화율을 함께 계산하여 결과의 크기를 해석한다. 차이가 연구 질문에서 의미 있는 수준인지 추가로 판단해야 한다.
[개념 6] 효과 크기와 실질적 의미에 대한 옳은 설명 3개를 고르시오.
정답: ②, ④, ⑤ — 차이의 크기와 실제 중요성을 함께 해석한다.

7 불확실성과 변동성

결과는 하나의 고정값이 아닐 수 있다

개념 상세 설명: 표본 자료의 평균과 모델 결과에는 표본 선택, 측정 오차와 자연스러운 변동이 포함된다. 평균뿐 아니라 표준편차, 범위와 반복 결과를 함께 제시하면 불확실성을 더 정직하게 표현할 수 있다.

import statistics

measurements = [9.8, 10.2, 9.9, 10.4, 9.7, 10.0]

mean_value = statistics.mean(measurements)
std_value = statistics.stdev(measurements)
data_range = max(measurements) - min(measurements)

print("평균:", round(mean_value, 2))
print("표준편차:", round(std_value, 2))
print("범위:", round(data_range, 2))
print("해석: 평균 주변에 측정값의 변동이 존재한다.")
코드 상세 설명: 평균만 제시하지 않고 표준편차와 범위를 함께 계산한다. 결과의 대표값과 변동성을 동시에 확인할 수 있다.
[개념 7] 불확실성과 변동성에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ⑤ — 대표값과 함께 변동과 오차 가능성을 밝혀야 한다.

8 자료와 분석 방법의 한계

결론이 약해질 수 있는 조건을 밝힌다

개념 상세 설명: 한계는 연구가 틀렸다는 뜻이 아니라 결론을 해석할 때 고려해야 할 조건이다. 표본 수, 표본의 대표성, 결측값, 측정 정확도, 변수 누락과 분석 방법의 가정 등을 구체적으로 밝힌다.

limitations = {
    "표본": "한 학급 20명만 포함되어 대표성이 제한됨",
    "기간": "일주일 자료만 사용하여 계절 변화가 반영되지 않음",
    "측정": "자가 보고식 집중도 점수라 주관적 오차가 가능함",
    "변수": "수면 시간과 수업 난이도를 포함하지 못함"
}

for category, detail in limitations.items():
    print(f"[{category}] {detail}")
코드 상세 설명: ‘자료가 부족하다’처럼 모호하게 쓰지 않고 표본, 기간, 측정과 변수 측면에서 구체적으로 한계를 기록한다.
[개념 8] 연구 한계에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ④ — 한계는 결론의 적용 조건과 불확실성을 구체화한다.

9 일반화와 적용 범위

누구에게 어디까지 적용할 수 있는가

개념 상세 설명: 일반화는 표본에서 얻은 결과를 더 넓은 대상에 적용하는 것이다. 표본이 모집단을 얼마나 잘 대표하는지, 장소·시간·연령과 조건이 다른 상황에서도 결과가 유지되는지 검토해야 한다.

sample_info = {
    "학교": "A고등학교",
    "학년": "1학년",
    "학생수": 24,
    "측정기간": "7일"
}

print("표본 정보:", sample_info)
print("적용 가능한 범위: 동일 학교 1학년의 유사한 기간에 우선 적용")
print("주의: 다른 지역, 학년, 계절에 일반화하려면 추가 검증이 필요함")
코드 상세 설명: 표본의 장소, 대상과 기간을 명시하고 결과가 우선 적용될 수 있는 범위를 제한적으로 표현한다.
[개념 9] 일반화와 적용 범위에 대한 옳은 설명 3개를 고르시오.
정답: ②, ③, ⑤ — 표본과 조건에 맞게 적용 범위를 제한해야 한다.

10 재검증과 개선 방향

한계를 다음 연구 계획으로 바꾼다

개념 상세 설명: 개선 방향은 단순히 ‘더 연구한다’고 쓰는 것이 아니라 무엇을, 어떻게, 왜 보완할지 제시해야 한다. 표본 확대, 기간 연장, 변수 추가, 측정 도구 개선, 다른 분석 방법 적용과 독립 자료 재검증을 포함할 수 있다.

improvement_plan = [
    {"문제": "표본이 작음", "개선": "3개 학급으로 표본 확대"},
    {"문제": "측정 기간이 짧음", "개선": "4주 동안 반복 측정"},
    {"문제": "수면 변수 누락", "개선": "수면 시간 자료 추가"},
    {"문제": "한 분석 방법 사용", "개선": "상관분석과 회귀분석 결과 비교"},
    {"문제": "동일 자료만 사용", "개선": "다른 시기의 독립 자료로 재검증"}
]

for item in improvement_plan:
    print(item["문제"], "→", item["개선"])
코드 상세 설명: 발견한 한계와 대응하는 개선 행동을 연결한다. 개선 후에는 같은 연구 질문을 다시 분석해 결과의 일관성을 확인해야 한다.
[개념 10] 재검증과 개선 방향에 대한 옳은 설명 3개를 고르시오.
정답: ①, ④, ⑤ — 한계를 보완하는 실행 가능한 계획과 재검증이 필요하다.
