🧩 학습할 핵심 개념 10개

1 연구 질문

탐구의 방향

개념 상세 설명: 연구 질문은 무엇을 어떤 조건에서 어떤 자료로 확인할지 드러나는 문장이다. “교실 환경은 어떤가?”보다 “환기 전후 교실 CO2 농도는 얼마나 달라지는가?”가 측정과 비교에 적합하다.

target = "교실"
condition = "환기 전후"
variable = "CO2 농도"
question = f"{condition} {target}의 {variable}는 얼마나 달라지는가?"
print(question)
코드 상세 설명: 대상, 조건, 측정 변수를 문자열로 조합하여 구체적인 연구 질문을 만든다. 질문만 읽어도 비교 대상과 측정값이 보이도록 하는 것이 중요하다.
[개념 1] 좋은 연구 질문의 조건으로 옳은 것 3개를 고르시오.
정답: ①, ③, ⑤ — 연구 질문은 구체적이고 측정 가능하며 실행 가능한 범위여야 한다.

2 가설

검증 가능한 예상

개념 상세 설명: 가설은 연구 질문에 대한 잠정적이고 검증 가능한 예상이다. 단순한 희망이나 확신이 아니라 자료에 의해 지지되거나 반박될 수 있어야 한다.

independent = "환기 시간"
dependent = "CO2 농도"
hypothesis = f"{independent}이 길어질수록 {dependent}는 낮아질 것이다."
print(hypothesis)
코드 상세 설명: 독립변인과 종속변인을 문장에 넣어 예상 관계를 표현한다. “낮아질 것이다”처럼 변화 방향을 넣으면 방향성 가설이 된다.
[개념 2] 가설에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ④ — 가설은 잠정적이며 검증과 반증 가능성을 가진다.

3 독립변인

원인 후보

개념 상세 설명: 독립변인은 연구자가 변화시키거나 집단을 나누는 기준으로 삼는 변인이다. 환기 시간, 학습 방법, 조도 수준 등이 독립변인이 될 수 있다.

ventilation_minutes = [0, 5, 10, 15]
for minute in ventilation_minutes:
    print("독립변인 값:", minute, "분")
코드 상세 설명: 환기 시간을 여러 수준으로 설정했다. 독립변인은 연구 질문에 따라 직접 조작하거나 이미 존재하는 조건을 비교할 수 있다.
[개념 3] 독립변인에 대한 옳은 설명 3개를 고르시오.
정답: ②, ③, ⑤ — 독립변인은 원인 후보이며 비교하거나 조작하는 기준이다.

4 종속변인

관찰되는 결과

개념 상세 설명: 종속변인은 독립변인의 변화에 따라 달라질 것으로 예상되어 측정하는 결과 변인이다. CO2 농도, 반응 시간, 정확도, 만족도 점수 등이 종속변인이 될 수 있다.

co2_values = [1350, 1100, 850, 720]
for i, value in enumerate(co2_values):
    print(i, "단계 종속변인:", value, "ppm")
코드 상세 설명: 각 조건에서 측정한 CO2 농도를 결과값으로 출력한다. 종속변인은 단위와 측정 방법이 명확해야 비교할 수 있다.
[개념 4] 종속변인에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ④ — 종속변인은 결과로 측정하며 방법과 단위가 필요하다.

5 통제변인

공정한 비교

개념 상세 설명: 통제변인은 독립변인 외에 종속변인에 영향을 줄 수 있어 가능한 한 같게 유지하는 조건이다. 교실 크기, 측정 위치, 학생 수, 측정 시간대 등을 통제하면 비교의 타당성이 높아진다.

controls = {
    "교실 크기": "동일",
    "측정 위치": "교실 중앙",
    "측정 시간": "수업 시작 10분 후"
}
for item, rule in controls.items():
    print(item, ":", rule)
코드 상세 설명: 통제할 조건과 유지 방법을 딕셔너리로 정리한다. 모든 변인을 완벽히 통제하기 어려운 경우에는 차이를 기록하고 한계로 설명해야 한다.
[개념 5] 통제변인에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ⑤ — 통제변인은 다른 영향을 줄여 비교의 타당성을 높인다.

6 조작적 정의

개념을 측정값으로

개념 상세 설명: 조작적 정의는 집중도, 만족도, 효율성처럼 추상적인 개념을 실제 측정 가능한 기준으로 바꾸는 것이다. 예를 들어 집중도를 ‘10분 동안 과제 화면을 벗어난 횟수’로 정의할 수 있다.

concept = "집중도"
operational_definition = "10분 동안 과제 화면을 벗어난 횟수"
print(concept, "=", operational_definition)
코드 상세 설명: 추상적 개념과 측정 기준을 연결했다. 조작적 정의는 연구자가 무엇을 어떻게 측정했는지 다른 사람이 이해하도록 돕는다.
[개념 6] 조작적 정의에 대한 옳은 설명 3개를 고르시오.
정답: ②, ③, ④ — 조작적 정의는 측정 기준을 구체화하고 재현성을 높인다.

7 방향성 가설

변화 방향 예상

개념 상세 설명: 방향성 가설은 독립변인이 증가하거나 감소할 때 종속변인이 어느 방향으로 변할지 예상한다. “조도가 높아질수록 독서 정확도가 증가할 것이다”처럼 관계의 방향을 제시한다.

light = [100, 300, 500]
predicted_accuracy = [72, 84, 91]
for x, y in zip(light, predicted_accuracy):
    print(f"조도 {x}lx → 예상 정확도 {y}%")
코드 상세 설명: 조도 증가에 따라 예상 정확도가 증가하는 방향을 자료 형태로 표현한다. 실제 결과는 예상과 다를 수 있으며 그 차이도 중요한 탐구 결과이다.
[개념 7] 방향성 가설에 대한 옳은 설명 3개를 고르시오.
정답: ①, ③, ⑤ — 방향성 가설은 근거를 바탕으로 변화 방향을 예상하고 검증한다.

8 반증 가능성

틀릴 수 있어야 검증 가능

개념 상세 설명: 반증 가능성은 어떤 결과가 나오면 가설이 지지되지 않는다고 판단할 수 있는 조건이 존재한다는 뜻이다. “좋은 기술은 언제나 좋다”처럼 어떤 결과도 설명할 수 있는 문장은 과학적 검증이 어렵다.

hypothesis = "환기 시간이 길수록 CO2 농도는 낮아진다"
result = "환기 시간과 CO2 농도 사이에 일관된 감소가 없음"
supported = "감소" in result and "없음" not in result
print("가설 지지 여부:", supported)
코드 상세 설명: 실제 결과가 예상 방향과 일치하는지 단순하게 점검한다. 연구에서는 통계 분석과 반복 측정으로 더 신중하게 판단해야 한다.
[개념 8] 반증 가능성에 대한 옳은 설명 3개를 고르시오.
정답: ②, ④, ⑤ — 과학적 가설은 틀릴 가능성이 열려 있어야 한다.

9 측정 척도와 단위

자료의 언어

개념 상세 설명: 측정 척도와 단위는 변인을 어떤 값으로 표현할지 정하는 기준이다. 온도는 ℃, CO2는 ppm, 만족도는 1~5점, 반응 시간은 초처럼 명확히 정해야 자료를 비교할 수 있다.

measurements = {
    "온도": "℃",
    "CO2 농도": "ppm",
    "만족도": "1~5점",
    "반응 시간": "초"
}
for variable, unit in measurements.items():
    print(variable, "단위:", unit)
코드 상세 설명: 변인과 측정 단위를 연결해 기록한다. 단위가 다르면 변환 방법을 명시하고, 설문 척도는 각 점수의 의미를 분명히 해야 한다.
[개념 9] 측정 척도와 단위에 대한 옳은 설명 3개를 고르시오.
정답: ①, ②, ④ — 단위, 척도 의미, 측정 도구의 한계를 함께 확인해야 한다.

10 질문·가설·변인의 정합성

설계의 연결

개념 상세 설명: 정합성은 연구 질문, 가설, 변인, 측정 방법이 서로 같은 관계를 가리키는 상태이다. 질문에는 환기 시간과 CO2가 있는데 가설은 온도와 집중도를 다루면 설계가 어긋난다.

question_vars = {"환기 시간", "CO2 농도"}
hypothesis_vars = {"환기 시간", "CO2 농도"}
print("정합성:", question_vars == hypothesis_vars)
코드 상세 설명: 질문과 가설에 포함된 변인 집합이 같은지 비교한다. 실제 점검에서는 방향, 대상, 조건, 측정 방법까지 함께 확인한다.
[개념 10] 질문·가설·변인의 정합성에 대한 옳은 설명 3개를 고르시오.
정답: ②, ③, ⑤ — 질문, 가설, 변인, 측정 방법이 같은 탐구 관계를 가리켜야 한다.

📚 학습과 관련한 탐구 보고서 주제 예시 5개

1. 환기 시간에 따른 교실 이산화탄소 농도 변화

연구 질문: 환기 시간이 길어질수록 교실 CO2 농도는 얼마나 감소하는가?

가설: 환기 시간이 길어질수록 CO2 농도는 낮아질 것이다.

2. 화면 밝기가 글 읽기 정확도에 미치는 영향

연구 질문: 화면 밝기 수준에 따라 글 읽기 정확도와 반응 시간은 달라지는가?

가설: 지나치게 낮거나 높은 밝기보다 중간 밝기에서 정확도가 높을 것이다.
