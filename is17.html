6. 학습할 코드 1개

project = {
    "title": "교실 공기질 알림 시스템",
    "problem": "수업 중 이산화탄소 농도 상승을 즉시 알기 어렵다.",
    "method": "센서 자료를 수집하고 기준값에 따라 상태를 분류했다.",
    "result": "환기 필요 구간을 자동으로 표시할 수 있었다.",
    "limitation": "한 교실의 짧은 기간 자료만 사용했다.",
    "improvement": "여러 교실과 계절 자료를 추가할 계획이다."
}

presentation_order = [
    ("문제", "problem", 20),
    ("방법", "method", 25),
    ("결과", "result", 25),
    ("한계", "limitation", 15),
    ("개선", "improvement", 15)
]


def create_summary(project, order):
    lines = [f'주제: {project["title"]}']
    total_seconds = 0

    for label, key, seconds in order:
        lines.append(f'{label}: {project[key]}')
        total_seconds += seconds

    lines.append(f'예상 발표 시간: {total_seconds}초')
    return "\n".join(lines)


summary = create_summary(project, presentation_order)
print(summary)
코드 요소상세 설명
project프로젝트의 제목, 문제, 방법, 결과, 한계, 개선 방향을 딕셔너리로 구조화한다.
presentation_order발표 순서, 딕셔너리 키, 예상 시간을 튜플로 저장한 리스트이다.
create_summary프로젝트 정보와 발표 순서를 입력받아 발표용 요약문을 만든다.
lines발표에 사용할 문장을 순서대로 저장한다.
total_seconds각 부분의 시간을 누적해 전체 발표 시간을 계산한다.
for label, key, seconds튜플의 세 값을 각각 발표 항목명, 딕셔너리 키, 시간으로 분리한다.
"\n".join(lines)여러 문장을 줄바꿈으로 연결해 하나의 발표문으로 만든다.

확장 활동: 청중 유형, 데이터 출처, 윤리적 고려, 핵심 메시지 항목을 추가하여 과학 발표용·학부모 설명용·교내 전시용 요약문을 각각 만들어 본다.
